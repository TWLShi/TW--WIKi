## 4. 等级判定矩阵

```
警情等级 = f(规模维度, 风险维度, 时间维度, 空间维度)
```

| 等级 | 触发条件 | 出车规模 |
|------|----------|----------|
| 一般 | 单点，影响范围小 | 1-2中队 |
| 较大 | 扩散风险，需多队协同 | 3-5中队 |
| 重大 | 人员伤亡，重大财产 | 6+中队 |
| 特别重大 | 灾难性，后果极其严重 | 全力处置 |
