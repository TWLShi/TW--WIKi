# 个人任务编排交互看板

**标签**：#任务编排 #交互看板 #Kanban #ClaudeCode #Subagents
**位置**：03_SystemDesign/

---

## 核心功能

| 功能 | 说明 |
|------|------|
| **编排** | 建立任务、委派给 Subagents |
| **实时追踪** | 拖拽调整、反馈循环 |
| **视觉 Kanban** | 交互式任务管理 |

---

## 主流工具（2026）

| 工具 | 特点 |
|------|------|
| **Vibe Kanban** | 一键启动 `npx vibe-kanban`，支持拖拽、指派 Subagent |
| **claude-code-kanban** | 监听任务文件，实时更新 Pending→In Progress→Done |
| **KANBAII** | AI-native，Teams 模式，成本追踪 |
| **Claude-Code-Kanban-Automator** | 视觉建立任务 → 自动执行 → 反馈更新 |

---

## 看板字段

| 字段 | 说明 |
|------|------|
| Backlog | 待办 |
| This Week / In Progress | 进行中 |
| Review | 待审核 |
| Done | 已完成 |

---

## FireCommand 指挥看板

| 字段 | 说明 |
|------|------|
| 接警待处理 | 新事件进入 |
| 灾情分析 | Situation Analyst 处理中 |
| 资源调派 | Resource Dispatcher 处理中 |
| 执行中 | 指挥方案执行 |
| 反馈闭环 | 现场实时反馈 |
| 已完成 | 事件结束 |

---

## 建立步骤

1. **先用 `/tasks` 熟悉内建功能**
2. **安装可视化看板**：`npx vibe-kanban` 或 `npx claude-code-kanban`
3. **自定义 Skill 实现自动编排**：定义 Kanban Orchestrator Skill
4. **应用到 FireCommand**：指挥事件作为大任务，每个阶段作为子任务卡片
