# 价值点提取——RAG增强接警槽位填充方案

**来源**：接警场景槽位填充 + 噪声SLU + Graph RAG/混合RAG研究分析
**版本**：V1.0

---

## 核心价值点

### 1. 细微事实识别提升
- **细微事实类型**（低洼/地下室/儿童/明火vs浓烟/程度）上下文依赖强、噪声敏感，纯蒸馏易遗漏
- RAG检索本体实例+历史案例+标注样本，为转录片段提供精确上下文
- **预期提升**：槽位F1额外+3-10%

### 2. 置信度校准改善
- 纯小模型过自信问题（overconfidence），传统softmax概率与真实准确率不匹配
- RAG提供外部证据支持度作为校准信号：强相关→提升置信，冲突/无匹配→降低置信
- **ECE改善8-11%**，Graph RAG可计算多跳证据支持度

### 3. 推荐实施路径
| 阶段 | 内容 |
|------|------|
| 主线 | 人工标注 + 蒸馏小模型（当前最稳） |
| 并行 | 本体+标注样本构建Graph RAG索引（Neo4j/Faiss+KG） |
| 部署 | 小模型主路径 + 低置信触发RAG（延迟增加100-300ms可控） |
| 协同 | 标注时增加"证据片段"/"可检索span"标签 |

### 4. 研究支持
| 研究 | 适用性 |
|------|--------|
| RASU | SLU任务intent/slot预测提升 |
| LA-RAG | 噪声/方言场景ASR后实体识别 |
| Graph RAG | 结构化关系密集领域，faithfulness优于Vector RAG |
| NAACL | Noise-Aware Calibration降低过自信 |

### 5. 输出示例
```
位置：XX路地下室（置信0.95，检索到3条类似低洼案例支持）
被困：2人儿童（置信0.78，因烟火程度证据中等）
```

### 6. 适用场景选择
- **纯蒸馏优先**：资源有限，追求快速上线
- **轻量Graph RAG早期引入**：追求高精度，愿意增加前期投入
