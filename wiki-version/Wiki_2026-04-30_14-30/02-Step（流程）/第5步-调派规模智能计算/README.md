## 3. N_total 公式

```
N_total = N_base + α_sub + β_struct + G_special + M_support + Δ_confidence + Δ_scene
```

| 因子 | 说明 | 权重 |
|------|------|------|
| N_base | 基础车辆数 | 固定值 |
| α_sub | 子场景系数 | 0.5-1.5 |
| β_struct | 建筑结构系数 | 0.8-1.2 |
| G_special | 特殊危险品系数 | 1.0-2.0 |
| M_support | 增援系数 | 0-1.0 |
| Δ_confidence | 置信度调整 | ±0.2 |
| Δ_scene | 现场态势调整 | ±0.3 |

## 4. 三档方案

| 方案 | 规模系数 | 适用场景 |
|------|----------|----------|
| 保守档 | 0.7×N_total | 信息不足、初步研判 |
| 标准档 | 1.0×N_total | 正常情况 |
| 增强档 | 1.3×N_total | 高风险、复杂场景 |
