# 1.3 场景画像库

场景画像库定义的是"系统理论上应该知道什么"。

---

## 核心概念

**场景画像**：在某个细分场景下，系统为了判断风险、定级和调派，理论上应该获得的一组关键字段。

---

## 场景画像库组织结构

```
场景画像库/
├── 通用要素/                        # 所有场景通用要素模板
├── 12类典型救援场景画像/             # 按灾害类型分类的12个典型场景
│   ├── 建筑火灾/                     # 7类建筑火灾
│   │   ├── 普通住宅/
│   │   ├── 高层建筑/
│   │   ├── 大型商业综合体/
│   │   ├── 地下建筑/
│   │   ├── 厂房仓库/
│   │   ├── 古建筑文物/
│   │   └── 城中村自建房/
│   ├── 工业与特殊场所火灾/           # 2类工业火灾
│   │   ├── 化工企业/
│   │   └── 石油储罐/
│   ├── 交通运输火灾/                 # 1类交通火灾
│   │   └── 新能源汽车/
│   └── 其他救援/                     # 2类其他救援
│       ├── 水域救援/
│       └── 地质灾害与建筑坍塌/
└── 6类聚类场景画像/                  # 按接警模式分类的6个聚类
    ├── 聚类1-水域灾害类/
    ├── 聚类2-普通火灾类/
    ├── 聚类3-复杂空间与高危结构类/
    ├── 聚类4-人员密集类/
    ├── 聚类5-重点单位类/
    └── 聚类6-极端紧急类/
```

## 6类聚类场景画像

根据`场景画像聚类知识.docx`，场景画像分为六个聚类，作为问题集分层设计基础：

| 聚类 | 名称 | 问题集继承 | 说明 |
|------|------|------------|------|
| 聚类1 | 水域灾害类 | L0 + 水灾专项 | 暴雨内涝/河道涨水/地下室进水 |
| 聚类2 | 普通火灾类 | L0 + L1 | 住宅/商铺/普通建筑（默认主场景） |
| 聚类3 | 复杂空间与高危结构类 | L0 + L1 + 高危结构专项 | 高层/地下/厂房/化工 |
| 聚类4 | 人员密集类 | L0 + L1 + 人员密集专项 | 学校/医院/商场/车站 |
| 聚类5 | 重点单位类 | L0 + L1 + 重点单位专项 | 政府机关/机房/危化品单位 |
| 聚类6 | 极端紧急类 | L0 最小必要信息集 | 报警人慌乱/通话极短 |

**继承关系**：
```
L0（必须）→ L1（标准火灾）→ L2（场景专项）
```

## 通用要素模板

所有场景都需要采集的通用要素：

| 要素类别 | 关键字段 | 说明 |
|----------|----------|------|
| **位置要素** | district, community, detailedAddress, floor | 定位警情发生地 |
| **警情要素** | incidentType, sceneType | 判断警情类型 |
| **火情要素** | fireStage, fireScale, smokeColor | 判断灾害程度 |
| **人员要素** | trappedCount, hasInjury, severity | 人员救援优先级 |
| **建筑要素** | buildingType, totalFloors, fireFloor | 影响救援策略 |
| **危险源要素** | hasDangerousItem, hazardousMaterial | 安全风险评估 |
| **通道情况** | evacuationRouteClear, roadAccess | 救援可达性 |
| **联系要素** | callerLocation, canContact | 保持联络 |
