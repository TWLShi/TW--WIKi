# 通用要素模板

通用要素模板定义所有警情场景都需要采集的基础要素。

---

## 通用要素总表

| 要素类别 | 关键字段 | 数据类型 | 说明 |
|----------|----------|----------|------|
| **位置要素** | district | string | 行政区划 |
| | community | string | 小区/社区名称 |
| | detailedAddress | string | 详细地址 |
| | street | string | 街道/路名 |
| | landmark | string | 地标 |
| | floor | string | 楼层 |
| | roomNumber | string | 房号 |
| **警情要素** | incidentType | enum | 警情类型 |
| | sceneType | enum | 场景类型 |
| **人员要素** | trappedCount | int | 被困人数 |
| | trappedLocation | string | 被困位置 |
| | hasInjury | boolean | 是否有人员伤亡 |
| | severity | string | 严重程度 |
| **火情要素** | fireStage | enum | 火灾阶段 |
| | fireScale | enum | 火势大小 |
| | smokeColor | enum | 烟雾颜色 |
| | smokeOdor | string | 烟雾气味 |
| **建筑要素** | buildingType | enum | 建筑类型 |
| | totalFloors | int | 总层数 |
| | fireFloor | int | 起火楼层 |
| | hasBasement | boolean | 是否有地下室 |
| **危险源** | hasDangerousItem | boolean | 是否有危险品 |
| | hazardousMaterial | string | 危险物质类型 |
| **通道情况** | evacuationRouteClear | boolean | 疏散通道是否畅通 |
| | exitBlocked | boolean | 出口是否被堵 |
| | stairwaySmoke | boolean | 楼梯间是否有烟 |

---

## L0层要素（必须获取）

L0层是接警最初阶段，必须获取的核心要素：

| 要素 | 字段 | 说明 | 影响 |
|------|------|------|------|
| 位置与警情 | l0.root | 位置和警情类型识别 | 确定辖区和出警地址 |
| 位置补充 | l0.pos | 具体到楼栋单元楼层房号 | 精确定位 |
| 人员被困 | l0.risk | 被困人数和情况 | 判断是否需要同步医疗 |

---

## 要素获取优先级

```
L0（必须）
    ↓
L1（标准问题）
    ↓
L2（场景专项）
```

| 层级 | 问题数 | 触发条件 |
|------|--------|----------|
| L0 | 3-4个 | 所有来话必问 |
| L1 | 7个 | 火灾场景标准问题 |
| L2 | 5-9个 | 特定场景专项问题 |
