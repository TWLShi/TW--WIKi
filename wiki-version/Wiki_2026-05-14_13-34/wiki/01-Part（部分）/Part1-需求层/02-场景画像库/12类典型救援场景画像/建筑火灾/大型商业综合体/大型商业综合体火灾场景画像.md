# 大型商业综合体火灾场景画像

大型商业综合体火灾是建筑火灾中的高危类型，具有人员密集、功能复杂、火灾蔓延快等特点。

---

## 场景定义

| 属性 | 值 |
|------|------|
| 场景编号 | T12-3 |
| 一级场景 | 建筑火灾 |
| 二级场景 | 大型商业综合体 |
| 风险等级 | 极高 |
| 调派类型 | 大规模人员+医疗联动 |

---

## 核心特征

- **人员高度密集**：顾客、员工众多
- **功能复杂**：商场、餐饮、娱乐多种业态
- **空间大**：中庭、走廊连通，蔓延快
- **可燃物多**：商品、装修材料多样
- **次生灾害风险**：踩踏、烟雾中毒

## 场景画像要素

### 位置要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| district | string | 是 | 行政区划 |
| unitName | string | 是 | 商场名称 |
| detailedAddress | string | 是 | 详细地址 |
| fireFloor | string | 是 | 着火楼层 |
| tenantType | string | 否 | 业态类型 |

### 人群要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| estimatedPeopleCount | int | 是 | 现场大概人数 |
| crowdDensity | string | 是 | 人群密度 |
| evacuationStatus | string | 是 | 疏散状态 |

### 火情要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| fireStage | enum | 是 | 火灾阶段 |
| fireScale | enum | 是 | 火势 |
| smokeSpread | string | 是 | 烟雾蔓延方向 |

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
