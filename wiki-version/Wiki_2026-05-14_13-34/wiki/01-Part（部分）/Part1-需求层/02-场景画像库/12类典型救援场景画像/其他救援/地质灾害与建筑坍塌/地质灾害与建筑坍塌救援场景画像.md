# 地质灾害与建筑坍塌救援场景画像

地质灾害与建筑坍塌救援是其他救援中的高危类型，涵盖地震、泥石流/山体滑坡、建筑坍塌等场景。

---

## 场景定义

| 属性 | 值 |
|------|------|
| 场景编号 | T12-12 |
| 一级场景 | 其他救援 |
| 二级场景 | 地质灾害与建筑坍塌 |
| 风险等级 | 极高 |
| 调派类型 | 重型救援+多部门联动 |

---

## 核心特征

- **突发性强**：地震、坍塌突然发生
- **人员埋压多**：可能有大量人员被困
- **救援难度大**：需要重型装备和专业技术
- **次生灾害风险**：余震、二次坍塌
- **多部门联动**：需要医疗、救援、建设等多部门

---

## 场景画像要素

### 位置要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| district | string | 是 | 行政区划 |
| locationName | string | 是 | 地点名称 |
| detailedAddress | string | 是 | 详细地址 |
| landmark | string | 否 | 地标 |

### 事故类型要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| disasterType | string | 是 | 灾害类型：地震/滑坡/坍塌 |
| magnitude | string | 否 | 地震震级（地震时） |
| collapseArea | string | 是 | 坍塌范围/面积 |
| hasAftershock | boolean | 否 | 是否有余震（地震时） |

### 人员要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| buriedCount | int | 是 | 埋压人数 |
| trappedCount | int | 是 | 被困人数 |
| rescueUrgency | string | 是 | 救援紧迫程度 |

### 建筑要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| buildingType | string | 是 | 建筑类型 |
| collapseType | string | 是 | 坍塌类型：整体/局部 |
| hasCollapseRisk | boolean | 是 | 是否有二次坍塌风险 |

### 环境要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| hasFireRisk | boolean | 是 | 是否有火灾 |
| hasGasLeak | boolean | 是 | 是否有燃气泄漏 |
| hasHazardousMaterial | boolean | 是 | 是否有危化品泄漏 |
| dustStatus | string | 否 | 粉尘情况 |
