# 人员密集场所场景画像

人员密集类包括学校、医院、商场、影院、酒店、车站、养老院、大型活动现场、培训机构、写字楼高峰时段等场所。

---

## 场景定义

| 属性 | 值 |
|------|------|
| 场景编号 | S1-9 |
| 一级场景 | 火灾类（S1） |
| 二级场景 | 人员密集场所 |
| 风险等级 | 高 |
| 调派类型 | 集中调派（大量人员装备） |

---

## 核心特征

与普通火灾相比，人员密集类最重要的不是"火有多大"，而是：
- **有多少人**：人员规模评估
- **人群是否恐慌**：踩踏、疏散压力
- **逃生通道是否受阻**：出口是否被堵
- **是否有老人、儿童、病患、行动不便人员**：特殊群体救援
- **是否需要立即组织疏散**：应急响应启动

## 场景画像要素

### 位置要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| district | string | 是 | 行政区划 |
| community | string | 是 | 场所名称 |
| detailedAddress | string | 是 | 详细地址 |
| buildingType | string | 是 | 场所类型：学校/医院/商场/酒店/车站 |
| floor | string | 否 | 着火楼层 |
| crowdType | string | 是 | 人员类型：学生/患者/顾客/员工/旅客 |

### 人群要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| estimatedPeopleCount | int | 是 | 现场大概人数 |
| evacuationStatus | string | 是 | 疏散状态：已疏散/进行中/未疏散 |
| crowdDensity | string | 是 | 人群密度：拥挤/一般/疏散中 |
| exitBlocked | boolean | 是 | 主要出口是否被堵 |
| hasPanic | boolean | 否 | 人群是否恐慌 |

### 人员风险要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| trappedCount | int | 是 | 被困人数 |
| trappedLocation | string | 是 | 被困位置（区域/楼层） |
| hasVulnerablePeople | boolean | 是 | 是否有老人/儿童/病人 |
| hasInjury | boolean | 是 | 是否有人受伤 |
| hasTrampleRisk | boolean | 是 | 是否有踩踏风险 |

### 通道要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| evacuationRouteClear | boolean | 是 | 疏散通道是否畅通 |
| emergencyExitCount | int | 否 | 应急出口数量 |
| elevatorStatus | string | 否 | 电梯是否停运 |

### 协调要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| hasCoordinator | boolean | 是 | 现场是否有负责人/保安/老师 |
| coordinatorContact | string | 否 | 负责人联系方式 |

## 人员密集细分场景

| 三级场景 | 描述 | 关键缺失要素 |
|----------|------|--------------|
| S1-9-1 | 商场火灾 | 人数规模、疏散状态、出口情况 |
| S1-9-2 | 学校火灾 | 学生人数、是否有老师组织、疏散难度 |
| S1-9-3 | 医院火灾 | 患者数量、病房位置、轮椅/病床人员 |
| S1-9-4 | 酒店火灾 | 住客人数、楼层高度、是否有外籍人员 |
| S1-9-5 | 大型活动现场火灾 | 活动规模、人员类型、是否有组织方 |

## 问题集继承结构

```
人员密集类 = L0 + L1 普通火灾 + 人员密集专项问题集
```

### L0层（必须）

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| l0.root | "您好，这里是119消防指挥中心！请问您遇到什么紧急情况？在什么位置？" | incidentType, sceneType, district, community |
| l0.pos | "能再说一下具体位置吗？具体到几栋几单元几楼几号？" | detailedAddress, landmark |
| l0.risk | "是否有人被困？有多少人？" | trappedCount, trappedLocation |

### L1层（普通火灾扩展）

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| l1.fire.1 | "烟是什么颜色？浓不浓？" | smokeColor, smokeOdor |
| l1.fire.2 | "火是刚着起来，正在蔓延，还是已经烧得很大？" | fireStage, fireScale |
| l1.fire.3 | "起火位置大概在哪里？" | fireFloor, burningMaterial |
| l1.fire.4 | "这里是什么类型的建筑？" | buildingType |
| l1.fire.5 | "一共有多少层？起火在第几层？" | totalFloors, fireFloor |
| l1.fire.6 | "楼梯、走廊或出口有没有被烟火堵住？" | evacuationRouteClear, exitBlocked |
| l1.fire.7 | "现场有没有人正在灭火？是否已经断电断气？" | hasInitialFirefighting, powerCut |

### 人员密集专项问题

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| c.1 | "现场大概有多少人？还有多少人没有撤离？" | estimatedPeopleCount, evacuationStatus |
| c.2 | "现场主要是什么人员？学生、患者、老人、顾客还是员工？" | crowdType |
| c.3 | "被困人员主要在哪个区域或楼层？" | trappedLocation, trappedCount |
| c.4 | "现在人员是否正在疏散？疏散是否顺利？" | evacuationStatus, evacuationRouteClear |
| c.5 | "主要出口和安全通道是否畅通？" | evacuationRouteClear, exitBlocked, emergencyExitCount |
| c.6 | "现场是否混乱？有没有拥挤、摔倒或踩踏情况？" | crowdDensity, hasInjury, hasTrampleRisk |
| c.7 | "是否已经有人受伤或吸入浓烟？" | hasInjury |
| c.8 | "现场有没有负责人、保安、老师或医护人员可以配合？" | hasCoordinator, coordinatorContact |

## 关键缺失判断

**公式**：关键缺失要素 = 场景画像要求 - 报警人已提供

**示例**：报警人说"XX商场三楼着火了，里面还有很多人没出来"

| 已提供 | 缺失（按优先级） |
|--------|------------------|
| ✅ 位置（XX商场三楼） | ❌ 人数规模 |
| ✅ 火情存在 | ❌ 疏散状态 |
| ✅ 有人被困 | ❌ 出口是否被堵 |
| | ❌ 是否有踩踏风险 |
| | ❌ 是否有负责人可协调 |
