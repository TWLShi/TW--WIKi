## 1. 核心公式汇总

### 1.1 规模计算公式
```
推荐车辆数 = round(基础车辆数 × 总系数)
总系数 = 建筑系数 × 风险系数 × 时间系数 × 面积系数（上限3.5）
单中队出车率 ≤ 70%
```

### 1.2 混合判定公式
```
final_score = 0.45 × rule_score + 0.55 × ml_score
```

### 1.3 路由优先级公式
```
priority_score = (
    风险权重 × 0.55 +
    信息缺失度 × 0.25 +
    时间敏感度 × 0.15 +
    历史同类案例匹配度 × 0.05
)
```

## 2. Multi-Agent架构（核心设计）

### 2.1 四Agent职责与权重

| Agent | 职责 | 权重 | 输入 | 输出 |
|-------|------|------|------|------|
| 感知Agent | 场景理解 + 事实基础 | 15% | 画像+模拟结果 | 场景标签+历史Top5 |
| 决策Agent | 战术规划 + 力量分工 | **40%** | 感知输出 | 主攻方向+小组分工 |
| 执行Agent | 可执行性验证 + 指令生成 | 20% | 决策输出 | 时间线+指令模板 |
| 预测Agent | 态势预测 + 风险预警 | 25% | 全局+决策输出 | 火势蔓延+次生灾害 |

### 2.2 仲裁机制（5阶段）
```
1. 结果标准化（统一JSON Schema）
      ↓
2. 交叉验证（检测冲突，阈值>0.25标记冲突）
      ↓
3. 加权投票（决策40%、预测25%、执行20%、感知15%）
      ↓
4. LLM仲裁（当分歧>0.3时触发）
      ↓
5. 二次验证（送回执行+预测再校验，通过率<85%强制人工）
```

### 2.3 仲裁阈值配置
```
自动通过阈值：冲突分 < 0.25
强制人工阈值：冲突分 > 0.45 或 置信度 < 0.78
高风险场景（4级/危化品/高层）→ 自动提升人工确认优先级
```

### 2.4 动态权重矩阵（可配置）
```
普通住宅：Decision 35% / Prediction 30%
化工场景：Prediction 40% / Execution 30%
高层建筑：Execution 35% / Decision 35%
```

## 3. 故障处理机制（Fault Tolerance）

### 3.1 故障分级

| 等级 | 典型场景 | 处理策略 | 恢复时间 | 人工通知 |
|------|----------|----------|----------|----------|
| 轻度 | 单Agent超时、置信度低 | 重试1次+缓存/默认值 | <5秒 | 否 |
| 中度 | 2个Agent失败、仲裁冲突大 | 降级模式（关闭预测Agent） | 5-15秒 | 记录警告 |
| 重度 | 3个以上Agent失败 | 纯规则模式+强制人工确认 | <10秒 | 是（高亮） |
| 致命 | 框架崩溃、核心服务不可用 | 全局降级为人工模式 | 立即 | 是（推送值班长） |

### 3.2 Fallback策略
```
Perception失败 → 使用第3步静态画像
Decision失败 → 使用规则引擎默认战术
Execution失败 → 使用简化时间线模板
Prediction失败 → 使用保守预测（最坏情况假设）
```

## 4. 约束校验机制（7步核心）

### 4.1 五维校验
```
1. 资源可用性校验（实时车辆状态）
2. 时间窗约束（ETA是否满足等级要求）
3. 特殊环境约束（城中村/窄巷/高层/化工）
4. 规模合法性约束（最小/最大车辆限制）
5. 政策与预案约束（跨区调派权限/重大活动限制）
```

### 4.2 校验状态与分流
```
PASS → 触发第8步（反向模拟）
WARNING → 带警告继续第8步
FAIL → 回退到第5步或第6步重新计算
```

---

## 5. 规模计算模型（5步核心）

### 5.1 输入聚合
```
- 第4步最终等级 + 置信度
- 第3步完整多维画像
- 实时资源池状态
- 全局配置（季节/区域政策/特殊预案）
```

### 5.2 三档方案
```
最小档（保守） / 推荐档（平衡） / 最大档（积极）
```

### 5.3 异常处理

| 异常场景 | 处理逻辑 | 优先级 |
|----------|----------|--------|
| ML模型不可用 | 回退到纯规则计算 | 高 |
| 资源极度紧张 | 强制输出最小档+跨区联动建议 | 最高 |
| 计算系数异常（>3.5或<0.8） | 截断处理+人工复核 | 高 |
| 画像数据缺失 | 使用默认值+降低置信度 | 中 |
| 全局平衡优化失败 | 回退到本地最优方案 | 中 |
