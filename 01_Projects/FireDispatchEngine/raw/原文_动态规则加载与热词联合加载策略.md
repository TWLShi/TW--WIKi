**来源**：119消防接警机器人（火灾场景）
**版本**：V1.0

## 1. 动态规则加载核心优势

| 优势 | 说明 |
|------|------|
| 精准性提升 | 只加载与当前画像匹配规则，避免无关规则干扰 |
| 性能优化 | 规则库O(n)→O(k)，降低推理延迟 |
| 可解释性增强 | 可清晰说明"本次激活了3条规则" |
| 适应性强 | 支持多场景，规则可持续迭代新增 |

### 风险与挑战
- 规则冲突风险 → 解决：规则分组+优先级+SHACL验证
- 加载延迟 → 解决：预构建规则索引
- 规则覆盖不全 → 解决：大模型Teacher持续迭代
- 过度过滤 → 解决：设置保底规则集

## 2. 规则分组设计

### 按场景分组
- fire_rules / openlock_rules / waspnest_rules / general_rules

### 按要素分组
- location_rules / ignition_rules / smoke_development_rules / building_structure_rules / risk_assessment_rules

### 按优先级分组
- high_priority（生命风险）
- medium_priority（财产风险）
- low_priority（分流建议）

## 3. 联合加载逻辑框架

```python
def joint_load_rules_and_hotwords(current_profile):
    activated_rules = []
    hotwords = []
    
    # 起火部位 = 厨房
    if current_profile.get("ignition_point") == "厨房":
        activated_rules.extend([
            "kitchen_combustion_rule",
            "residential_smoke_development",
            "child_risk_in_kitchen"
        ])
        hotwords.extend(get_hotwords_by_group("kitchen_fire"))
        boost_weight(hotwords, 1.8)
    
    # 受害人 = 儿童
    if "儿童" in current_profile.get("victim_category", ""):
        activated_rules.append("child_priority_rescue_rule")
        hotwords.extend(get_hotwords_by_group("child_risk"))
        boost_weight(hotwords, 2.0)
    
    # 保底规则
    activated_rules.append("general_person_risk_rule")
    
    return activated_rules, select_top_k(hotwords, k=30)
```

## 4. 具体场景联合加载示例

### 示例1：已知"起火部位=厨房"+"受害人=儿童"
**激活规则**：
- kitchen_combustion_rule
- residential_smoke_development
- child_priority_rescue_rule
- kitchen_child_high_risk_rule

**最终热词子集**：
- 高权重(4.0-5.0)：油锅着火、抽油烟机、厨房浓烟、孩子在厨房、儿童烧伤
- 中权重(2.5-3.5)：燃气灶、油烟很大、从厨房蔓延
- 过滤：厂房化工品、仓库、高空马蜂窝

### 示例2：已知"场所用途=厂房"+"燃烧物=化工品"
**激活规则**：
- industrial_hazard_rule
- chemical_leak_explosion_rule
- steel_structure_collapse_rule
- toxic_smoke_evacuation_rule

**最终热词子集**：
- 高权重(4.5-5.0)：化工品泄漏、爆燃风险、毒烟扩散、钢结构着火
- 过滤：油锅着火、卧室、儿童

### 示例3：已知"烟火发展=猛烈"+"建筑结构=高层住宅"
**激活规则**：
- intense_fire_command_rule
- high_rise_vertical_spread_rule
- high_rise_evacuation_rule

**最终热词子集**：
- 高权重(4.5-5.0)：火势凶猛、猛烈燃烧、垂直蔓延、整层燃烧
- 过滤：初期着火、油锅着火、厂房钢结构
