# 火灾画像要素完整问题集

**来源**：119消防接警机器人（火灾救援场景）
**版本**：V1.0

---

## 1. 火灾画像要素获取排序（推荐优先级）

1. **位置信息**（最高优先级，必须先明确）
2. **起火部位 + 受困位置**（核心判断火情与人员危险）
3. **火势大小 + 燃烧范围 + 烟的情况**（颜色、浓淡、范围、重点部位）
4. **烟火的发展状况**（判断火情发展趋势与蔓延风险）
5. **燃烧物 + 危险物品**
6. **建筑结构**（影响逃生、灭火难度和指挥决策）
7. **受困状态**（人数、状态）
8. **场所性质、单位、业主、报警人情况**

---

## 2. 火灾画像完整要素清单

### 核心要素

| 要素 | 说明 |
|------|------|
| **位置信息** | 详细地址 + 区域特征（高层、地下室、厂房等） |
| **起火部位** | 厨房、卧室、电气间、仓库、车辆等 |
| **受困位置** | 被困人员所在具体位置（是否与起火部位同一区域） |
| **火势大小** | 初期/中等/猛烈/无法控制 |
| **燃烧范围** | 单点/单间/多间/整层/多层/整栋 |
| **烟的情况** | 颜色(黑烟/白烟/黄烟)、浓淡(浓烟/薄烟)、范围、重点部位 |
| **烟火的发展状况** | 发展阶段（初始/发展/猛烈/衰减/熄灭）、蔓延速度、爆燃风险 |
| **燃烧物** | 电器、家具、油类、塑料、木材、化工品 |
| **危险物品** | 燃气罐、油漆、化学品、电池、爆炸物 |
| **建筑结构** | 建筑类型(住宅楼/厂房/仓库)、结构特征(钢结构/防火分区)、楼层数 |

### 辅助要素
- **受困状态**：人数、人员类型（老人/儿童/伤员）、状态
- **场所性质**：住宅、厂房、仓库、商场、医院、学校
- **单位/业主信息**：名称、联系方式
- **报警人情况**：身份（业主/邻居/路人）、联系电话

---

## 3. 烟火发展状况5个状态

| 状态 | 名称 | 典型描述 |
|------|------|----------|
| 1. 初始阶段 | 刚着火 | 刚着火、火苗很小、刚冒烟 |
| 2. 发展阶段 | 发展蔓延 | 正在蔓延、火势增大、已经烧大 |
| 3. 猛烈阶段 | 猛烈燃烧 | 火势很大、猛烈燃烧、控制不住 |
| 4. 衰减阶段 | 衰减控制 | 火势减弱、已被控制、正在扑灭 |
| 5. 熄灭阶段 | 已熄灭 | 已经熄灭、火已灭、没有明火 |

## 4. 完整问题集（按优先级排序）

### 第1层：位置信息
1. 请告诉我火灾的具体位置和详细地址？
2. 具体在哪个小区、楼栋、单元或楼层？

### 第2层：火情严重度判断
3. 火势现在怎么样？是刚开始着火、火势中等，还是已经很大、控制不住？
4. 烟的情况如何？是什么颜色？浓不浓？主要从哪个部位冒出来？
5. 烟火现在的发展情况怎么样？是刚着火、正在蔓延、火势猛烈、已经被控制，还是已经扑灭？

### 第3层：人员风险评估
6. 是否有人被困？被困的具体位置和起火部位是否在同一区域？
7. 被困人员有多少人？主要是老人、孩子还是伤员？他们的状态如何？

### 第4层：现场细节补充
8. 起火位置大致在哪？是厨房、卧室、电气间还是其他地方？
9. 燃烧范围有多大？是单个房间、多个房间、整层楼，还是已经蔓延到多层？
10. 主要燃烧的是什么东西？有电器、家具、油类还是化工品？
11. 现场是否有危险物品？比如燃气罐、油漆、化学品或易爆物？
12. 这里是什么类型的建筑？是高层住宅、厂房、仓库还是老旧楼房？楼层数多少？

### 第5层：辅助信息核实
13. 这里是住宅、单位还是其他场所？单位名称是什么？
14. 业主或负责人联系方式方便提供吗？
15. 您是业主、邻居还是单位员工？您的联系电话是多少？

### 第6层：确认与处置建议
16. 我已记录：位置[XX]、火势[XX]、烟的情况[XX]、烟火发展[正在蔓延]、受困[XX人]、建筑结构[高层住宅]……建议立即派消防车。请确认是否正确？
