## 模块二：红色路径（火灾确认）

| 场景 | 话术 |
|------|------|
| 居民火灾 | 消防救援，请讲明：是居民家还是单位着火？在哪个地址？ |
| 化工火灾 | 消防救援，着火物质是什么？有无危险品储存？ |
| 高速火灾 | 您在哪个方向（南向北）？最近下匝道？路边K标多少？ |

## 模块三：蓝色路径（其他救助）

| 场景 | 话术 |
|------|------|
| 社会救助 | 具体是什么事？有危险吗？ |
| 开锁 | 请核实：是反锁还是有老人小孩需要紧急开锁？ |
| 蛇进家 | 蛇还在吗？是什么品种？ |

## 模块四：模糊警情核实

> "请核实：您现在是看到火了，还是只看到烟？"

| 回答 | 结果 |
|------|------|
| 看到火 | 确认火灾（C=1.0，强制红色路径） |
| 只看到烟 | 疑似火灾（C=0.7，侦察调派） |
| 都不是 | 路径切换至蓝色抢险/救助 |
