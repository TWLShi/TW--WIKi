## 4. 匹配规则

```
最优队站 = argmin(Distance(Incident, Station) × LoadFactor(Station))
```

| 约束 | 说明 |
|------|------|
| 距离约束 | 队站到现场距离最短 |
| 负载约束 | 单中队出车率≤70% |
| 类型约束 | 危化品需特种车辆 |
| 时间约束 | 考虑高峰期路况 |
