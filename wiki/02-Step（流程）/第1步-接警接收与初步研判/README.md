# Step 1：接警接收与初步研判

**DIKW层级**：Data（数据层）
**核心职责**：119接警、ASR转写、初步标签分类

---

## 1. Step概述

本Step是整个消防接处警指挥调度流程的起点，负责接收报警电话、进行语音转写（ASR）、提取关键槽位信息，并完成初步的警情标签分类。

---

## 2. 核心内容

- 119报警电话接收
- ASR语音转写与降噪
- 关键槽位提取（地点、时间、类型等）
- 初步警情标签分类
- 警情时间线建立
