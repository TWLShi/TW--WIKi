警情跟踪指挥是值班值守人员采用远程监控、现场图传、语音通话、图文传输等手段，对灾情发展态势、处置行动及其关联信息不间断收集、分析和汇总的过程，并依据火警与应急救援分级，结合火情灾情危险程度，赶赴现场组织指挥火灾扑救和执行应急救援任务的行为。

## "三段指挥"工作机制

"三段指挥"是对灭火救援警情处置全过程开展的跟踪指挥行动，主要包括接警调度、途中行进和到场展开三个阶段，即调度指挥、途中指挥和现场指挥。加强"三段指挥"工作，是以"打赢制胜"为根本目标，从形式、内容、流程到应用上对指挥模式进行全面升级，着力解决灾情研判不到位、力量调派不精准、初控措施不得当、战术运用不合理、装备使用不高效、安全防护不规范等问题。

### 工作主体

"三段指挥"模式由总队、支队两级指挥中心、全勤指挥部捆绑运行。两级指挥中心大厅要增设指挥长、作战助理、通信助理席位，配备相关通信联络与辅助决策器材设备。全勤指挥部作战指挥人员必须随身携带手持电台，24小时保持通信畅通，遇有敏感警情立即到指挥中心了解相关情况，介入"三段指挥"工作。重要节日、重大安保期间，全勤指挥部作战指挥人员24小时在指挥中心值守。

### 分工协作

- **指挥中心**：侧重调度指挥和途中指挥，突出收集警情、调足力量、安全提示、后台派送等环节，为全勤指挥部的决策指挥提供有力信息支撑
- **全勤指挥部**：侧重现场指挥，结合指挥中心的先期处置和现场视频传输图像，研判评估灾情态势是否可控、参战力量是否充足、作战行动是否安全，持续做好辅助决策、风险管控和技战术运用等跟踪指挥

### 响应流程

- **一级警情**：支队指挥中心和全勤指挥部发挥属地主体作用，落实"三段指挥"，随时做好总队指挥中心备询
- **二级警情**：支队全勤指挥部遂行出动、到场指挥，总队指挥中心和全勤指挥部介入"三段指挥"
- **三级以上警情**：总队全面协调，辖区支队属地负责，增援支队积极配合，打造上下同频、左右衔接的指挥网络

总队、支队全勤指挥部实行双班备勤制度，遇有重大警情，主班赶赴现场时，备班须迅速到指挥中心接替应急值守工作，直至主班人员返回交接。

### 工作任务

**调度指挥阶段的主要任务**：
1. **接警询问**：最大限度从报警人处获取有关信息（灾情地点、燃烧物质、起火部位、火势蔓延、建筑毗邻、人员被困和伤亡等），引导报警人逃生自救，接应消防车辆，保持通信联络
2. **调派力量**：根据灾情类型，按照"就近调派、多点调派、等级调派"的原则，最短时间调集辖区消防站、增援消防站、多种形式消防队伍等首批力量携带有效装备到场处置
3. **联动部门**：视情联动协调应急、公安、医疗、电力、供水、燃气及社会专业力量协同作战

**途中指挥阶段的主要任务**：
1. **核实地址任务**：与各个参战力量联系，逐一核实警情地址和作战任务，通过营区监控监督参战车辆出动情况
2. **跟踪车辆动态**：利用车辆定位系统实时跟踪参战车辆行进动态，提示正确选择行车路线，保持电台联络畅通，及时打开图传设备
3. **研判灾情态势**：根据后续报警电话、视频监控图像、实时网络舆情、灾情对象预案等信息，研判分析灾情发展态势，视情启动应急响应机制
4. **推送处置要点**：及时通报现场情况，推送处置对策和安全要点，提示预作作战行动部署和安全防护措施
5. **预见性增援调派**：综合考虑事故类型、交通道路、水源分布、气象条件、作战实力、建筑环境等信息，提前调派专业处置队、远程供水编成、战勤保障车辆、大型工程机械等特种力量和装备

**现场指挥阶段的主要任务**：
1. **到场展开提示**：灵活运用《初起火灾作战行动要诀》和《类型灾害事故初战处置行动规程》，提示首批到场力量落实初战控火救人措施
2. **获取现场信息**：督促将侦察询情和信息报送贯穿于灭火救援处置全过程，及时获取现场基本情况、人员被困伤亡、燃烧物质面积、灾情发展态势、战斗进展成果等重要信息
3. **视频辅助指挥**：通过单兵图传、4G布控球、无人机航拍等视频传输，实时了解现场态势和战斗行动，远程辅助指挥参战力量调整作战部署
4. **安全风险管控**：根据《常见风险种类与风险评估方法》，加强对现场作战安全风险的辨识评估、观察监测和预警提示
5. **作战态势标绘**：根据灾情现场的交通道路、消防水源、气象条件、建筑毗邻和力量编成等情况，围绕灾情对象进行态势标绘
6. **调整参战力量**：根据灾情发展态势和现场需求，视情对参战力量作出增援调派、集结待命、归建轮换等调整部署
7. **传达作战指令**：第一时间将上级部门和各级领导的作战指令传达至各参战力量，并抓好督办落实与信息反馈

### 工作要求

- **思想高度重视，精心部署谋划**："三段指挥"是总队首长提出的旨在提升队伍战斗力的创新举措，各级全勤指挥部指挥长要带领全体指战员高起点部署，高站位谋划，高标准落实
- **坚持问题导向，推动提质强能**：对照"三段指挥"工作任务，全面梳理作战指挥工作，重点从人员配备、能力培养、机制创新、作战成效等方面推动落实
- **督导一体推进，狠抓工作成效**：总队灭火救援指挥部将不定期采取视频抽查、实地检查、交叉互查等方式，每月对"三段指挥"工作落实情况进行视频调度

### 能力建设

坚持"六训"支撑，围绕岗位专业能力等级达标考评，适时组织岗位培训、下沉锻炼、业务考核，常态化开展桌面推演和案例复盘。升级智能接处警系统，深化"多种形式联动调度平台"应用，依托消防救援"一张图"建设，聚合各类应急数据资源。

## 一体化作战指挥程序

### 火灾扑救基本程序

**指挥中心**：受理警情 → 接警询问（基本/必要/分类要素）→ 力量调派 → 跟踪提示

**基层队站**：执勤备战 → 响应出动 → 途中行进 → 到场准备 → 安全防护 → 灾情侦察 → 设立警戒 → 疏散救人 → 破拆排烟 → 火场供水 → 灭火战斗 → 现场清理 → 恢复战备

**全勤指挥部**：一体化值守 → 信息搜集 → 辅助决策 → 远程指挥 → 灾情研判 → 遂行出动 → 现场指挥 → 安全管控 → 信息报送

### 抢险救援基本程序

类似火灾扑救程序，增加了控灾排险、部门协同处置、专业指导救助等环节。

## 典型警情调度指挥要点

共24类典型警情，每类涵盖接警询问、调派力量、联动部门、应急响应、搜集信息、安全提示六个方面：

### 火灾扑救类（12类）

**1. 多层建筑火灾**
- **接警询问**：火势大小、燃烧物质、层高、面积、建筑结构、消防通道位置、被困人员数量及位置
- **调派力量**：水罐消防车、压缩空气泡沫车、抢险救援车、举高消防车
- **安全提示**：注意触电防范和燃气泄漏，防踏空坠落；外围出水控火防蔓延；火灾扑救按先控制后消灭原则

**2. 高层建筑火灾**
- **接警询问**：建筑层数、高度、使用性质、起火部位、消防设施运行情况、人员被困数量及位置
- **调派力量**：登高平台消防车、高喷消防车、超高层供水车、压缩空气泡沫车、抢险救援车
- **安全提示**：严禁乘坐普通电梯，逐层搜索人员；选择着火层下一层或下两层为进攻起点；充分利用室内消火栓出水控火，启动消防泵和防排烟系统

**3. 地下建筑火灾**
- **接警询问**：建筑功能结构、储存物质种类数量、出入口数量和位置、电器电路状况
- **调派力量**：排烟照明车、灭火机器人、高倍数泡沫车、大功率水罐车
- **安全提示**：编组行动，铺设救生照明线，全程保持通信畅通；采取灌注高倍数泡沫灭火；慎重采取破拆或注水措施

**4. 大跨度建筑火灾**
- **接警询问**：建筑面积、结构形式、生产工艺、储存物质、着火部位
- **调派力量**：大功率水罐车、泡沫车、高喷车、远程供水系统、机器人
- **安全提示**：水枪阵地位于承重墙柱下方，车辆严禁靠近墙体；利用消防坦克或灭火机器人实施内部灭火

**5. 城市综合体火灾**
- **接警询问**：建筑结构、面积、层数、业态组成、起火部位、人员被困情况
- **调派力量**：大功率水罐车、高喷车、排烟照明车、抢险救援车、登高车
- **安全提示**：依托防火分区设置阵地，充分利用固定消防设施（喷淋/消火栓/防排烟）；分区域逐层搜索，开启应急广播疏散

**6. 危化品仓库火灾**
- **接警询问**：危化品种类、理化性质、储存方式、储量
- **调派力量**：干粉车、化学事故抢险车、防化洗消车、机器人、大功率水罐车
- **安全提示**：车辆停靠上风方向，车头朝外；严格警戒，划分重危/轻危/安全区；必须着全封闭防化服

**7. 油罐火灾**
- **接警询问**：油品介质、油罐类型结构、储量液位高度、有无保温层、邻近罐情况
- **调派力量**：石化专业处置队、高喷车、泡沫车、远程供水系统、机器人
- **安全提示**：防火防爆防毒防冻防灼伤，设置观察哨；优先启动固定/半固定消防设施；保持冷却不间断；大面积流淌火先围堵再扑灭

**8. 露天堆垛火灾**
- **接警询问**：堆垛数量、体积、占地面积、储存物资种类、毗邻情况
- **调派力量**：大功率水罐车、远程供水系统、工程机械
- **安全提示**：严禁贸然攀登垛顶，防止飞火切断退路；调集挖掘机翻垛配合扑救；注意堆垛坍塌伤人

**9. 电化学储能电站火灾**
- **接警询问**：电池类型、容量、接入电压、起火部位、有无有毒气体
- **调派力量**：大功率水罐泡沫车、高倍数泡沫车、机器人、有毒气体检测仪
- **安全提示**：严禁带电作业，严禁射水未过火电池模块；在技术人员指导下扑救，保持安全距离

**10. 高速铁路火灾**
- **接警询问**：列车信息、乘客数量、断电放电情况、疏散平台位置
- **调派力量**：泡沫车、抢险车、排烟车、照明车
- **安全提示**：确认列车停运和区间断电后方可行动；优先使用疏散平台进入线路

**11. 船舶火灾**
- **接警询问**：船舶位置、着火部位、船体结构、储存物质、有无危险品
- **调派力量**：举高喷射车、消防拖轮、泡沫车、水罐车
- **安全提示**：穿戴专用救生衣，严防坠海；注意船舶倾覆危险，不得盲目登船

**12. 涉疫场所火灾**
- **接警询问**：收治隔离人员数量、起火部位、建筑结构
- **调派力量**：水罐车、化学洗消车、抢险车
- **安全提示**：着全封闭防化服，最大限度减少进入隔离区域；归队前全面洗消

### 抢险救援类（12类）

**13. 建筑坍塌救援**
- **接警询问**：建筑结构类型、使用性质、坍塌部位及范围、被困人员数量
- **调派力量**：地震专业处置队、搜救犬、探测仪器、破拆工具组、工程机械
- **安全提示**：多角度监测，设安全员和观察哨；先稳固支撑再破拆救人；严禁在坍塌危险区域大量集结人员

**14. 水域救援**
- **接警询问**：水域深度、水流流速、被困人员数量与状态、有无潜水设备
- **调派力量**：冲锋舟、橡皮艇、水域救援专业队、潜水装具
- **安全提示**：五个决不允许（不穿救生衣不下水、不戴潜水装备不下水等）；控制船速，严禁超载

**15. 地下有限空间救援**
- **接警询问**：空间类型（井/池/管/坑）、人员被困数量及状况、有无有毒有害气体
- **调派力量**：抢险救援车、照明车、气体检测仪、移动供气源
- **安全提示**：全程检测有毒有害气体，双绳保护；未检测严禁盲目进入，必须佩戴空呼

**16. 交通事故救援**
- **接警询问**：车辆情况、运载物质、有无危化品泄漏、被困人员数量及伤势
- **调派力量**：抢险救援车、水罐车、破拆工具组
- **安全提示**：普通道路警戒距离>200米，高速>500米；先固定车辆再破拆，防止二次伤害

**17. 危化品运输事故**
- **接警询问**：罐体类型、装载物质名称及性质、泄漏相态、有无人员中毒
- **调派力量**：水罐车、防化洗消车、化学事故抢险车、堵漏器材
- **安全提示**：剧毒及无法确认的危化品不得盲目处置；必须在上风方向设置阵地

**18. 燃气泄漏救援**
- **接警询问**：泄漏相态（扩散/聚集/爆燃）、扩散范围、周边火源情况、有无人员被困
- **调派力量**：水罐车、泡沫车、气体检测仪、防爆装备
- **安全提示**：内着棉质内衣，皮肤不得外露；禁绝一切火源，防止产生火花

**19. 山岳救援**
- **接警询问**：遇险者位置、人数、伤情、地形地貌、气象条件
- **调派力量**：山岳救援专业队、绳索装备、无人机
- **安全提示**：确认地理坐标，绳索设备用保险；注意气象变化和次生灾害

**20. 地震灾害**
- **接警询问**：震中位置、震级、建筑物倒塌范围及数量、人员被困情况
- **调派力量**：地震救援专业队、搜救犬、破拆工具组、生命探测仪、工程机械
- **安全提示**：多角度监测建筑变化，设安全哨及时预警；注意余震和次生灾害

**21. 洪涝灾害**
- **接警询问**：雨情汛情、人员被困位置及数量、道路通行情况
- **调派力量**：龙吸水、远程供水系统、浮动泵、冲锋舟、橡皮艇
- **安全提示**：避开带电设备和线路；注意水流速度和暗流漩涡

**22. 森林火灾**
- **接警询问**：起火方位、燃烧植被种类、火场面积、风向风速、周边道路水源
- **调派力量**：森林灭火专业队、风力灭火机、水罐车、无人机
- **安全提示**：密切关注风向变化，遇陡坡山谷暂避锋芒；严防飞火下山进城（"山火下山进城"）

**23. 雨雪冰冻灾害**
- **接警询问**：重点道路桥梁结冰、建筑物倒塌受损、电力设施倒塌停电、人员受困主要区域和数量
- **调派力量**：根据气象预警适时调整战备等级，预置消防救援力量和战勤保障车辆以及御寒装备物资
- **安全提示**：车辆安装防滑链，开道车及时喷洒除雪剂。救援人员编队行进，配备防寒防冻装备及药品

**24. 大风天气警情**
- **接警询问**：短时间接报多点、多地、多类警情，以室外火警、建筑坍塌救援以及道路、树木、广告牌、高空坠物等清障救助警情为主
- **调派力量**：以支队为单位统筹调派，全员上岗、全席位开放，视情采取接警与处警分离模式
- **安全提示**：正确选择车辆停靠位置和进攻阵地，加强口罩、毛巾、护目镜等个人防护，远离不稳定建（构）筑物和树木，预防电线断落触电和火势突变被困
