消防接处警工作贯穿从受理119报警开始到消防救援力量行动结束返队的全过程，始终围绕着警情受理、力量调度和辅助指挥这一主线，是灭火救援行动的重要组成部分和首要环节。

## 接警用语规范

119接处警工作践行全心全意为人民服务宗旨，坚持"人民至上、生命至上"的理念，遵循"有警必接、依法受理、快速准确、规范灵活"的原则。接警员应当使用普通话，做到接听及时、用语规范、询问简洁、记录准确、指令清晰、专业高效。

### 工作态度规范

1. **精神饱满，情绪平稳**：不把个人情绪带到工作中，不受报警人情绪影响
2. **态度热情，用语规范**：礼貌用语开场，"请"字在先，主动告知受理结果
3. **积极引导，适当安抚**：主动耐心引导报警人讲清事件、地点等关键要素，缓解紧张情绪
4. **业务熟练，询问简明**：用语准确简练、切中关键，对新情况迅速报告
5. **诚恳服务，耐心周到**：批评性、建议性电话有礼有节，不与报警人争辩

### 首接用语

1. 统一使用首接用语："您好，天津市119"
2. 无人讲话时："您好，天津市119，请讲？"（重复两遍，间隔两秒）→ 仍无人讲话："您的电话没有声音，如需帮助请换部电话再打来，再见"
3. 群众误拨119时："您的电话号码拨错了，请核实后再拨"
4. 外省市警情："这里是天津市119，请您加区号拨打本地119报警"

### 询问用语

1. **地址**："请问地址在哪里？在哪个行政区？哪条马路？哪个小区/村镇/单位"
2. **灾情**："什么东西/哪个地方着火了？现场有明火有烟吗？单位是干什么的？大概面积是多少？现场火势怎么样？有没有危险物品？"
3. **人员情况**："现场有没有人员被困/伤亡？您出来了吗？需要帮您联系120吗？" — 如被困必须问清具体位置/房间/区域

### 引导用语

1. 地址不清："请您提供更详细的现场位置或附近明显的标志"
2. 声音小/口音重："请您讲普通话，语速慢些，声音大一点，再讲一遍"
3. 逻辑混乱："请您听我说，我问您答""您旁边还有其他人吗，方便让他接电话吗"
4. 被困提醒：可撤离→"您用湿毛巾捂住口鼻，在确保安全的情况下，撤离到安全区域"；无法逃离→"请您注意自身安全，选择安全区域（阳台、窗口）等待救援，与我们保持联系"
5. 物业管理人员报警："请抓紧启动固定消防设施，组织疏散人员，有新的情况，马上与我们联系"
6. 位置复杂："我们已派警，请您在路口或明显位置等待并指引消防车辆"

### 安抚用语

1. 慌乱紧张："请您别着急，现在听我说（不断重复），把地址告诉我，我才能把消防车派出去"
2. 不配合后续询问："消防车辆已派出，我现在问的情况不影响消防车辆出动，请您配合"
3. 楼上居民紧张报警时："请您放心，消防队正在灭火处置，我们马上安排现场消防人员联系您"
4. 催促消防队到场时："消防队已经赶赴现场，请您稍等片刻"
5. 占线等候时："因为报警电话繁忙，让您久等了，请您谅解"

### 解释用语

1. 不属于消防职责范围时："抱歉，您反映的情况，不属于119受理范围，请您与XX部门联系（提供联系方式）"
2. 咨询出警是否收费："请您放心，我们消防站出警不收取任何费用"
3. 无险情开门开锁类救助："抱歉，消防队没有专业的开锁工具，建议您联系备案的开锁公司"
4. 预约救助时（核实无险情）："抱歉，消防队无预约服务，请您在需要时拨打119"

### 结束用语

1. 正常结束："我们已派警，请您注意安全，保持手机畅通，再见"
2. 重复报警（已核实地点）："我们已接到报警，消防力量正在赶往现场，再见"
3. 报警人表示感谢时："不客气，这是我们的本职工作，有需要您再来电话，再见"

### 特殊情形话术

**受理投诉119接警员**：
> "很抱歉，我们的服务没有令您满意，感谢您提供的宝贵意见，您先别着急，您反映的情况我已详实记录，请留下您的联系方式和姓名，一会儿会有专门处理投诉的人员与您联系。"

**受理举报消防有关事宜**：
> "您好，我们是119接警部门，请您拨打消防救援总队总机电话，为您进行转接。" — 总机话务员转接至信访或其他相关处室/支队

**受理恶意骚扰来电**：
> "请您注意自己的言行，我们电话有全程录音功能，并能显示来电号码，提醒您的行为已经干扰到我们正常工作，请不要随意拨打119，恶意占用公共资源需要承担相应法律责任。"
>
> 来电人员依然不收敛："我们将把电话线路让给其他需要帮助的人，再见。"随后直接挂断。

### 接警工作禁忌规范

**六个严禁**：
1. 严禁长时间不应答或无故挂机
2. 严禁拒绝、推诿、扯皮现象
3. 严禁态度冷漠，语气生硬，情绪失控与报警人争吵
4. 严禁答非所问，不耐烦、急躁，催促报警人
5. 严禁出现辱骂、呵斥等不文明语言
6. 严禁擅自对外泄漏报警人及119接处警有关信息

**禁忌词语**：
- 禁止使用"喂！"
- 禁止使用"应该是""好像是""可能是"等模棱两可的词汇
- 禁止使用"哪？""什么地？""哪个区？"等不规范词语
- 报警人叙述停顿时，禁止使用"说""讲""接着说"，应使用"请讲""您请讲"
- 禁止使用"不可能"等否定词直接否定报警人，应耐心解释政策："我们掌握的政策是……"
- 没有听清时，禁止使用"什么""您说的是他么""嘛"等口语化词汇，应使用"抱歉，我没有听清，能麻烦您再重复一遍吗？"

**禁忌语气**：
- 禁止质问语气："我不是告诉你了吗？没听清？我给您再念一遍啊！"
- 禁止反问语气："你问我，我怎么知道？什么怎么样？然后呢？"
- 禁止命令性或不耐烦语气："我们就是个接警的部门。不可能，电话多少，我给您查一下。"
- 禁止侮辱性词语或嘲笑、责备语气："您要这么说，我也没办法。这是您的权利。"

## 警情受理类型

消防警情共分为"火灾扑救""应急救援""社会救助""公务执勤""反恐排爆"及"其他出动"六大类。

### 1. 火灾扑救

| 分类 | 细项 |
|------|------|
| 居民住户 | 低层居住建筑、多层居住建筑、高层居住建筑 |
| 重要场所 | 行政办公场所、文化教育场所、医疗卫生场所、交通枢纽场所、邮电通信场所 |
| 商业建筑 | 购物场所、娱乐场所、餐饮场所、旅游场所、纪念性场所、民生服务单位、体育设施场所、门店商铺 |
| 厂房仓库 | 生产加工车间、物资仓库、化工单位、石油化工装置、危化品仓库、露天堆垛、油/气储罐（库）、加油（气）站、建筑工地、闲置厂房、其他场所、五供单位 |
| 交通工具 | 机动车、非机动车、轨道车辆、船舶、飞行器 |
| 室外设施 | 广告牌、路灯、电线/变压器、燃气管线 |
| 山林原野 | 山火、绿化、荒草、垃圾及废弃物 |
| 其他 | 温室（大棚）、简易或临时建筑、畜禽饲养场所、手工小作坊、地下建筑 |

### 2. 抢险救援

| 分类 | 细项 |
|------|------|
| 交通事故 | 公路、铁路、船舶、公路铁路隧道、飞行器 |
| 燃气跑漏 | 城市燃气管道、室内燃气管线、危化品运输 |
| 危险化学品泄漏 | 压缩气体和液化气、易燃固体、易燃可燃液体、放射性物质、毒害品 |
| 群众遇险事件 | 电梯故障、设备故障救人、井下事故、高空事故、水域事故、触电事故、遇险被困、其他群众遇险 |
| 突发性事件 | 刑事事件、群体性事件、治安事件、爆炸及恐怖事件、核事故、其他突发性事件 |
| 自然灾害 | 气象灾害次生事故、地质灾害次生事故、海洋灾害次生事故、核与辐射事故、空难 |
| 建筑物垮塌事故 | 地面建筑、地下建筑、施工中建筑、火灾情况下倒塌 |
| 其他救援 | 机动车漏油、异味/异响、报警器预警、营救轻生人员、重大环境污染事件、突发公共卫生事件 |

### 3. 社会救助

开门开锁、救助（捕捉）动物、清障/监护、摘除蜂窝、取戒指、关闭居民水/气阀门、排水、救助病人、高空取物、送水、其他救助

### 4. 公务执勤

重大会议、大型文体活动、大型展览会、其他公务执勤

### 5. 反恐排爆

可疑物处置、生化事故处置、爆炸物处置、废炮弹处置、寻衅滋事、其他反恐排爆

### 6. 其他出动

误报火警、误报救援

### 火警分级判定标准

| 级别 | 颜色 | 判定条件（人员伤亡或被困情况） |
|:----:|:----:|------|
| 一级 | 绿色 | 无人员伤亡或被困 |
| 二级 | 蓝色 | 3人以下伤亡或5人以下被困 |
| 三级 | 黄色 | 3人以上5人以下伤亡或5人以上10人以下被困 |
| 四级 | 橙色 | 5人以上10人以下伤亡或10人以上30人以下被困 |
| 五级 | 红色 | 10人以上伤亡或30人以上被困 |

### 应急救援分级判定标准

与火警分级相同的人员伤亡/被困判定条件，适用于"高、低、大、化"各类型灾害事故。
