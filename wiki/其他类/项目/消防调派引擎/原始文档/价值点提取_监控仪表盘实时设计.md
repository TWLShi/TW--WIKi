# 监控仪表盘实时设计 - 价值点提取

**标签**：#监控仪表盘 #实时监控 #可视化 #WebSocket #ECharts
**提取日期**：2025-04-25
**来源**：消防调派智能系统-实时监控仪表盘设计v1.0.txt
**版本**：V1.0

---

## 1. 仪表盘设计目标

| 目标 | 说明 |
|------|------|
| 实时性 | 全链路1-12步状态秒级更新 |
| 可视化 | 一屏掌握全局态势 |
| 可操作性 | 支持快速干预（人工确认、方案调整） |
| 分层设计 | 总览 → 详情 → 异常钻取 |

---

## 2. 仪表盘布局（4宫格 + 侧边栏）

```
┌──────────────────────────────────────────────────────────────┐
│  顶部全局状态栏（固定）                                      │
├──────────────┬───────────────────────┬───────────────────────┤
│ 左侧地图视图 │  中间流程监控看板     │  右侧资源与告警       │
│  (GIS实时)   │  (1-12步管道)         │  (车辆/中队/负载)     │
├──────────────┴───────────────────────┴───────────────────────┤
│ 底部活跃警情列表 + 异常待办                                 │
└──────────────────────────────────────────────────────────────┘
```

---

## 3. 五大功能模块

### 模块1：全局概览卡片（顶部）
- 当前活跃警情数
- 平均响应时间（接警→出车）
- AI整体置信度（今日平均）
- 资源负载率（全区车辆出动率）
- 高危警情（3/4级）数量
- 待人工确认数量（红色高亮）

### 模块2：GIS地图视图（左侧）
- 实时警情点位（颜色代表等级）
- 出动中队与车辆轨迹
- ETA热力圈
- 水源点、风险区域叠加
- 点击点位 → 右侧弹出该警情全链路状态

### 模块3：全链路流程监控看板（中间核心）
水平管道图展示1-12步实时状态：

```mermaid
flowchart LR
    A[1.接警解析] --> B[2.智能问询] --> C[3.画像构建] --> D[4.等级判定]
    D --> E[5.规模计算] --> F[6.车辆匹配] --> G[7.约束校验]
    G --> H[8.反向模拟] --> I[9.AI多智能体] --> J[10.人工确认]
    J --> K[11.方案下达] --> L[12.审计归档]

    classDef running fill:#22d3ee,stroke:#164e63
    classDef done fill:#4ade80,stroke:#166534
    classDef waiting fill:#fbbf24,stroke:#b45309
    classDef blocked fill:#ef4444,stroke:#991b1b
```

每个节点显示：
- 当前状态（运行中 / 已完成 / 等待 / 阻塞）
- 处理耗时
- AI置信度（第4、5、9步）
- 异常标记

### 模块4：资源与告警面板（右侧）
- **车辆状态分布**（饼图）：空闲 / 出勤 / 维修
- **中队负载Top5**
- **实时告警列表**（按严重程度排序）

### 模块5：活跃警情列表（底部）
- 警情ID / 时间 / 地址 / 等级 / 当前步骤 / 耗时 / 操作（快速干预按钮）

## 4. 技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端框架 | React + Ant Design Pro / Vue3 + Element Plus |
| 实时推送 | WebSocket + Server-Sent Events |
| 地图引擎 | Mapbox / Gaode Map |
| 数据源 | PostgreSQL + Redis + ClickHouse |
| 可视化 | ECharts + Mermaid + AntV G2 |

---

## 5. KPI指标

| 指标 | 说明 |
|------|------|
| 活跃警情数 | 当前处理中的警情 |
| 平均响应时间 | 接警到出车的时间 |
| AI整体置信度 | 今日平均置信度 |
| 资源负载率 | 车辆出动率 |
| 高危警情数 | 3/4级警情数量 |
| 待确认超时 | 需要人工确认的数量 |
