# 个人任务看板实现方式

**标签**：#任务看板 #Kanban #ClaudeCode #Subagents #Skills
**位置**：03_SystemDesign/

---

## 内建任务展示

| 功能 | 说明 |
|------|------|
| `/tasks` | 任务管理 UI，展示 Pending/In Progress/Completed/Failed |
| `Ctrl + T` | 快速切换显示/隐藏任务面板 |
| 状态实时更新 | Subagent 执行时终端显示状态行 |

---

## 看板字段设计

| 字段 | 说明 |
|------|------|
| **Backlog** | 待办 |
| **This Week / Today** | 本周/今日重点 |
| **In Progress** | 进行中 |
| **Review** | 待审核 |
| **Done** | 已完成 |

---

## 任务卡片信息

- 任务标题 + 简短描述
- 优先级（高/中/低，颜色标记）
- 所属 Subagent 或负责人
- 截止时间 / ETA
- 进度百分比或状态标签
- 链接到详细输出

---

## 建立步骤

| Level | 内容 |
|-------|------|
| **Level 1 基础** | 启用 Memory，建立 `tasks.md` 或 `personal-board.md` |
| **Level 2 建立 Skill** | 定义 Kanban Skill，Claude 自动读取并渲染看板 |
| **Level 3 结合 Subagents** | 主 Orchestrator 更新看板状态，Subagents 回报结果 |
| **Level 4 视觉增强** | Markdown + emoji 或生成 HTML dashboard |

---

## 视觉增强方式

| 方式 | 说明 |
|------|------|
| Markdown + emoji | 简单看板 |
| 本地 HTML | 带颜色、拖拽，桌面 dashboard |
| claude-code-kanban | 开源工具，终端任务转浏览器 Kanban |

---

## 消防指挥看板应用

| 字段 | 说明 |
|------|------|
| 事件编号 | 事件唯一标识 |
| 灾情等级 | 一般/较大/重大/特别重大 |
| 当前阶段 | 接警 → 分析 → 调派 → 执行 |
| 现场反馈状态 | 实时反馈更新 |
