# 1.3 场景画像库

场景画像库定义的是"系统理论上应该知道什么"。

---

## 核心概念

**场景画像**：在某个细分场景下，系统为了判断风险、定级和调派，理论上应该获得的一组关键字段。

---

## 通用要素模板

所有场景都需要采集的通用要素：

| 要素类别 | 关键字段 | 说明 |
|----------|----------|------|
| **位置要素** | 详细地址、楼栋、单元、楼层、房号、起火位置 | 定位警情发生地 |
| **火情要素** | 是否明火、火势大小、烟雾情况、是否蔓延 | 判断火灾程度 |
| **人员要素** | 是否有人被困、老人儿童孕妇病人情况、人员是否已疏散 | 人员救援优先级 |
| **建筑要素** | 是否高层、楼道是否畅通、电梯/消防通道情况 | 影响救援策略 |
| **危险源要素** | 是否有燃气罐、天然气、电动车电池、易燃物 | 安全风险评估 |
| **处置状态** | 是否已断电断气、是否有人尝试扑救 | 现场状态 |
| **联系要素** | 报警人位置、是否在现场、联系电话 | 保持联络 |

---

## 住宅厨房火灾场景画像示例

| 要素类别 | 关键字段 | 缺失影响 |
|----------|----------|----------|
| 位置要素 | 详细地址（省市区街道小区楼栋单元楼层房号） | 致命：无法到场 |
| 位置要素 | 起火位置（厨房具体位置） | 辅助：影响扑救方向 |
| 火情要素 | 是否有明火 | 致命：判断火灾真实性 |
| 火情要素 | 火势大小（烟大小、火高低） | 调派：决定派什么力量 |
| 火情要素 | 是否蔓延 | 调派：决定规模大小 |
| 人员要素 | 是否有人被困 | 致命：决定救援优先级 |
| 人员要素 | 老人/儿童/孕妇/病人情况 | 致命：影响医疗配置 |
| 危险源要素 | 是否有燃气罐 | 致命：爆炸风险 |
| 危险源要素 | 是否有天然气 | 致命：爆炸风险 |
| 危险源要素 | 是否有电动车电池 | 调派：特殊火灾类型 |
| 处置状态 | 是否已断电断气 | 辅助：安全确认 |
| 联系要素 | 报警人位置 | 辅助：保持联络 |

---

## 高层建筑火灾场景画像要点

相比普通住宅火灾，高层建筑额外需要：

| 要素类别 | 关键字段 | 缺失影响 |
|----------|----------|----------|
| 建筑要素 | 楼层层数（≥7层为高层） | 调派：决定举高车配置 |
| 建筑要素 | 电梯是否可用 | 调派：影响人员疏散路线 |
| 建筑要素 | 消防通道情况 | 致命：决定能否到达 |
| 火情要素 | 火灾楼层 | 致命：影响人员疏散 |
| 火情要素 | 竖向蔓延情况（烟囱效应） | 调派：决定隔离策略 |
| 人员要素 | 被困人员所在楼层 | 致命：决定救援优先级 |

## 场景画像库的组织方式

```
场景画像库/
├── 通用要素/                        # 所有场景通用要素
├── 住宅/                            # S1-1 住宅火灾
│   └── 住宅火灾场景画像.md
├── 水域灾害/                         # S4-1 水域灾害
│   └── 水域灾害场景画像.md
├── 复杂空间与高危结构/               # S1-2 高层/地下/厂房
│   └── 复杂空间与高危结构场景画像.md
├── 人员密集场所/                     # S1-9 商场/学校/医院
│   └── 人员密集场所场景画像.md
├── 重点单位/                         # S1-10 政府/机房/危化
│   └── 重点单位场景画像.md
├── 极端紧急/                         # L0-Mode 极简接警模式
│   └── 极端紧急场景画像.md
├── 高层建筑/                         # S1-2 高层建筑火灾
├── 工厂仓库/                         # S1-4 工厂仓库火灾
└── 车辆火灾/                         # S1-5 车辆火灾
```

## 六场景聚类体系

根据`场景画像聚类知识.docx`，场景画像分为六个聚类，作为问题集分层设计基础：

| 聚类 | 名称 | 问题集继承 |
|------|------|------------|
| 聚类1 | 水域灾害类 | L0 + 水灾专项 |
| 聚类2 | 普通火灾类 | L0 + L1 |
| 聚类3 | 复杂空间与高危结构类 | L0 + L1 + 高危结构专项 |
| 聚类4 | 人员密集类 | L0 + L1 + 人员密集专项 |
| 聚类5 | 重点单位类 | L0 + L1 + 重点单位专项 |
| 聚类6 | 极端紧急类 | L0 最小必要信息集 |
