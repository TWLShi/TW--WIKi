# 高层建筑火灾场景画像

高层建筑火灾是 S1 火灾类下的重要场景类型，包括高层住宅、写字楼、商务大厦等。

---

## 场景定义

| 属性 | 值 |
|------|------|
| 场景编号 | S1-2 |
| 一级场景 | 火灾类（S1） |
| 二级场景 | 高层建筑火灾 |
| 风险等级 | 高-极高 |
| 调派类型 | 举高车+特种装备 |

---

## 核心特征

- **7层及以上**高层建筑
- **烟囱效应明显**：竖向蔓延速度快
- **人员疏散难度大**：逃生时间有限
- **举高车需求**：需要云梯车、举高车
- **内部固定消防设施**：室内消火栓、自动喷淋

---

## 场景画像要素

### 位置要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| district | string | 是 | 行政区划 |
| community | string | 是 | 小区/大厦名称 |
| detailedAddress | string | 是 | 详细地址 |
| buildingType | string | 是 | 建筑类型：高层住宅/写字楼/商务大厦 |
| fireFloor | int | 是 | 火灾楼层 |
| totalFloors | int | 是 | 总层数 |
| unitNumber | string | 否 | 单元号 |

### 火情要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| fireStage | enum | 是 | 火灾阶段：初期/蔓延/大火 |
| smokeSpreadDirection | string | 是 | 烟气扩散方向：向上/向下/走廊 |
| stairwaySmoke | boolean | 是 | 楼梯间是否有烟 |
| smokeColor | enum | 是 | 烟雾颜色：白烟/黑烟 |

### 人员要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| trappedCount | int | 是 | 被困人数 |
| trappedLocation | string | 是 | 被困位置（楼层/区域） |
| hasVulnerablePeople | boolean | 否 | 是否有老人/儿童/病人 |

### 建筑设施要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| hasBalcony | boolean | 否 | 是否有阳台（逃生通道） |
| elevatorStatus | string | 是 | 电梯状态：可用/不可用/停运 |
| sprinklerStatus | string | 否 | 室内消防栓状态 |
| hasFireControlRoom | boolean | 否 | 是否有消防控制室 |

### 通道要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| evacuationRouteClear | boolean | 是 | 疏散通道是否畅通 |
| exitCount | int | 否 | 疏散出口数量 |
| roadAccess | boolean | 是 | 消防车能否到达楼下 |

## 高层建筑火灾细分场景

| 三级场景 | 描述 | 关键缺失要素 |
|----------|------|--------------|
| S1-2-1 | 高层住宅火灾 | 火灾楼层、被困位置、烟囱效应 |
| S1-2-2 | 写字楼火灾 | 楼层高度、人员密度、疏散状态 |
| S1-2-3 | 商务大厦火灾 | 建筑结构、地下空间、危化品 |

## 问题集继承结构

```
高层建筑火灾类 = L0 + L1 普通火灾 + 高危结构专项问题集
```

### L0层（必须）

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| l0.root | "您好，这里是119消防指挥中心！请问您遇到什么紧急情况？在什么位置？" | incidentType, sceneType, district, community |
| l0.pos | "能再说一下具体位置吗？具体到几栋几单元几楼几号？" | detailedAddress, fireFloor |
| l0.risk | "是否有人被困？有多少人？" | trappedCount, trappedLocation |

### L1层（普通火灾扩展）

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| l1.fire.1 | "烟是什么颜色？浓不浓？" | smokeColor, smokeOdor |
| l1.fire.2 | "火是刚着起来，正在蔓延，还是已经烧得很大？" | fireStage, fireScale |
| l1.fire.3 | "起火位置大概在哪里？" | fireFloor, burningMaterial |
| l1.fire.4 | "这里是什么类型的建筑？" | buildingType |
| l1.fire.5 | "一共有多少层？起火在第几层？" | totalFloors, fireFloor |
| l1.fire.6 | "楼梯、走廊或出口有没有被烟火堵住？" | evacuationRouteClear, stairwaySmoke |
| l1.fire.7 | "现场有没有人正在灭火？是否已经断电断气？" | hasInitialFirefighting, powerCut |

### 高危结构专项问题

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| h.1 | "烟是往楼上、楼下，还是向走廊扩散？" | smokeSpreadDirection |
| h.2 | "烟囱效应明显吗？楼梯间能看到烟吗？" | stairwaySmoke |
| h.3 | "楼内消防栓是否能用？水压正常吗？" | sprinklerStatus |
| h.4 | "消防车能不能开到楼下或门口？" | roadAccess |
| h.5 | "现场有没有物业、保安或企业负责人？" | hasCoordinator |
| h.6 | "电梯还能不能用？" | elevatorStatus |

## 关键缺失判断

**示例**：报警人说"XX大厦20楼着火了，楼梯都是烟，有人还在里面"

| 已提供 | 缺失（按优先级） |
|--------|------------------|
| ✅ 位置（XX大厦20楼） | ❌ 总层数 |
| ✅ 建筑类型（高层） | ❌ 火灾阶段 |
| ✅ 楼梯有烟（逃生受阻） | ❌ 燃烧物类型 |
| ✅ 人员被困 | ❌ 消防车可达性 |
| | ❌ 是否已断电 |
