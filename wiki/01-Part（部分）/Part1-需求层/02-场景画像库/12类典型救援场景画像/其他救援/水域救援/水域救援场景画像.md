# 水域救援场景画像

水域救援是其他救援中的重要类型，涵盖城市内涝、洪水急流救援、涉水车辆解困等场景。

---

## 场景定义

| 属性 | 值 |
|------|------|
| 场景编号 | T12-11 |
| 一级场景 | 其他救援 |
| 二级场景 | 水域救援 |
| 风险等级 | 高 |
| 调派类型 | 舟艇装备+防汛抗洪 |

---

## 核心特征

- **水患类型多样**：暴雨内涝、河流泛滥、山洪
- **二次伤害风险**：触电、溺水、缺氧
- **动态变化**：水位、水流变化快
- **救援装备特殊**：需要舟艇、潜水装备
- **高并发报警**：同一地点可能多人报警

---

## 场景画像要素

### 位置要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| district | string | 是 | 行政区划 |
| waterArea | string | 是 | 水域位置 |
| waterLocation | string | 是 | 受困位置 |
| landmark | string | 否 | 地标 |

### 水情要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| waterDepth | string | 是 | 水深（到人体部位） |
| waterLevelTrend | enum | 是 | 水位趋势：涨/稳/降 |
| waterFlowSpeed | enum | 是 | 水流速度 |
| disasterType | string | 是 | 灾害类型 |

### 人员要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| trappedCount | int | 是 | 被困人数 |
| hasDrowning | boolean | 是 | 是否有溺水 |
| hasVulnerablePeople | boolean | 是 | 是否有老弱病残 |

### 危险要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| hasElectrocutionRisk | boolean | 是 | 是否有触电风险 |
| hasGasLeak | boolean | 是 | 是否有燃气泄漏 |
| isUndergroundSpace | boolean | 是 | 是否为地下空间 |

### 救援要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| rescueAccessible | boolean | 是 | 救援是否可达 |
| needBoat | boolean | 是 | 是否需要舟艇 |
