# 城中村/自建房火灾场景画像

城中村/自建房火灾是建筑火灾中的特殊类型，具有消防通道差、密度高、人员复杂等特点。

---

## 场景定义

| 属性 | 值 |
|------|------|
| 场景编号 | T12-7 |
| 一级场景 | 建筑火灾 |
| 二级场景 | 城中村/自建房 |
| 风险等级 | 高-极高 |
| 调派类型 | 重点调派+多车联动 |

---

## 核心特征

- **建筑密度极高**：间距不足，火灾易蔓延
- **消防通道狭窄**：大型消防车难以进入
- **消防设施缺乏**：缺乏消火栓、灭火器
- **人员复杂**：租户多，对建筑不熟悉
- **逃生通道不足**：只有一个楼梯

## 场景画像要素

### 位置要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| district | string | 是 | 行政区划 |
| villageName | string | 是 | 城中村名称 |
| detailedAddress | string | 是 | 详细位置描述 |
| buildingType | string | 是 | 建筑类型：自建房/出租屋 |
| buildingHeight | int | 是 | 楼层数 |
| hasViolation | boolean | 否 | 是否有违建 |

### 火情要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| fireStage | enum | 是 | 火灾阶段 |
| fireScale | enum | 是 | 火势 |
| spreadDirection | string | 是 | 蔓延方向 |
| hasAdjacentRisk | boolean | 是 | 是否威胁相邻建筑 |

### 人员要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| trappedCount | int | 是 | 被困人数 |
| residentCount | int | 否 | 居住人数 |
| hasForeignWorkers | boolean | 否 | 是否有外来务工人员 |

### 通道要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| roadAccess | boolean | 是 | 消防车能否到达 |
| alleyWidth | string | 否 | 巷道宽度 |
| hasFireHydrant | boolean | 否 | 是否有消火栓 |
