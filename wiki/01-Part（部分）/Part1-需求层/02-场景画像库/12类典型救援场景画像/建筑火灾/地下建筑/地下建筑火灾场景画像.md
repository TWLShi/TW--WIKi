# 地下建筑火灾场景画像

地下建筑火灾是建筑火灾中的特殊类型，包括地下车库、地下室、地下商场、地铁站等，具有排烟困难、疏散困难等特点。

---

## 场景定义

| 属性 | 值 |
|------|------|
| 场景编号 | T12-4 |
| 一级场景 | 建筑火灾 |
| 二级场景 | 地下建筑 |
| 风险等级 | 高-极高 |
| 调派类型 | 排烟照明+特种装备 |

---

## 核心特征

- **空间密闭**：排烟困难，烟雾聚集
- **疏散受限**：楼梯是唯一疏散通道
- **视线差**：浓烟严重影响搜救
- **通信困难**：信号差，指挥受限
- **升温迅速**：烟气层下沉，温度快速上升

## 场景画像要素

### 位置要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| district | string | 是 | 行政区划 |
| community | string | 是 | 场所名称 |

| entryLocation | string | 是 | 地下空间入口位置 |
| exitCount | int | 是 | 出入口数量 |

### 火情要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| fireStage | enum | 是 | 火灾阶段 |
| fireLocation | string | 是 | 着火位置 |
| smokeColor | enum | 是 | 烟雾颜色 |
| smokeOdor | string | 是 | 烟气特征 |

### 人员要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| trappedCount | int | 是 | 被困人数 |
| trappedLocation | string | 是 | 被困位置 |
| evacuationStatus | string | 是 | 疏散状态 |

### 环境要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|

| ventilationStatus | string | 是 | 通风状态 |
| sprinklerStatus | string | 否 | 喷淋状态 |
