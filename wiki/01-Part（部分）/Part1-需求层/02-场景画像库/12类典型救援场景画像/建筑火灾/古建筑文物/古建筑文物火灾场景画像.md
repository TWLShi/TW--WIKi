# 古建筑/文物火灾场景画像

古建筑/文物火灾是建筑火灾中的特殊类型，具有文化保护价值高、灭火方式特殊等特点。

---

## 场景定义

| 属性 | 值 |
|------|------|
| 场景编号 | T12-6 |
| 一级场景 | 建筑火灾 |
| 二级场景 | 古建筑/文物 |
| 风险等级 | 极高 |
| 调派类型 | 特殊灭火+文物专家联动 |

---

## 核心特征

- **文化价值不可复制**：文物古建一旦损毁不可复原
- **灭火方式受限**：不能大量水喷淋，需专业灭火剂
- **结构老旧**：木结构为主，易燃烧蔓延
- **消防设施不足**：缺乏现代消防设施
- **周边环境复杂**：多位于老城区，通道狭窄

---

## 场景画像要素

### 位置要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| district | string | 是 | 行政区划 |
| unitName | string | 是 | 建筑/单位名称 |
| detailedAddress | string | 是 | 详细地址 |
| buildingType | string | 是 | 建筑类型：木构/砖木/历史建筑 |
| protectionLevel | string | 是 | 保护级别：国家级/省级/市级 |

### 火情要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| fireStage | enum | 是 | 火灾阶段 |
| fireLocation | string | 是 | 起火位置 |
| smokeColor | enum | 是 | 烟雾颜色 |
| spreadSpeed | string | 否 | 蔓延速度 |

### 文物要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| keyArtifacts | string | 否 | 重要文物位置 |
| has irreplaceableItems | boolean | 是 | 是否有不可移动文物 |
| evacuationPriority | string | 否 | 文物转移优先级 |

### 通道要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| roadAccess | boolean | 是 | 消防车能否到达 |
| historicalDistrict | boolean | 是 | 是否位于历史街区 |
