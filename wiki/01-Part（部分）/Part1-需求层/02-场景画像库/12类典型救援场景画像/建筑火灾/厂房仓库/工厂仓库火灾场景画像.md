# 工厂仓库火灾场景画像

工厂仓库火灾是 S1 火灾类下的重要场景类型，包括工业厂房、仓库、物流中心等。

---

## 场景定义

| 属性 | 值 |
|------|------|
| 场景编号 | S1-4 |
| 一级场景 | 火灾类（S1） |
| 二级场景 | 工厂仓库火灾 |
| 风险等级 | 高-极高 |
| 调派类型 | 特种装备+危化专家 |

---

## 核心特征

- **燃烧物复杂**：原料、成品、危化品混杂
- **火势蔓延快**：货架密集，火灾荷载大
- **供水需求大**：需要大流量供水车辆
- **可能涉及危化品**：需要防化专家和特种装备
- **建筑结构多样**：大跨度厂房、钢结构仓库

---

## 场景画像要素

### 位置要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| district | string | 是 | 行政区划 |
| community | string | 是 | 园区/工厂名称 |
| detailedAddress | string | 是 | 详细地址 |
| buildingType | string | 是 | 建筑类型：厂房/仓库/物流中心 |
| buildingStructure | string | 否 | 结构类型：钢结构/混凝土/大跨度 |

### 火情要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| fireStage | enum | 是 | 火灾阶段：初期/蔓延/大火 |
| fireScale | enum | 是 | 火势：小/中/大 |
| smokeColor | enum | 是 | 烟雾颜色：白烟/黑烟 |
| smokeOdor | string | 否 | 烟气特征：刺鼻/无味 |

### 危险源要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| hasDangerousItem | boolean | 是 | 是否有危化品 |
| hazardousMaterialList | string | 否 | 危化品清单 |
| hasExplosionRisk | boolean | 是 | 是否有爆炸风险 |
| burningMaterial | string | 否 | 燃烧物类型 |

### 建筑设施要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| totalFloors | int | 否 | 层数（单层/多层） |
| hasSprinkler | boolean | 否 | 是否有自动喷淋 |
| sprinklerStatus | string | 否 | 喷淋系统状态 |
| hasFireControlRoom | boolean | 否 | 是否有消防控制室 |

### 通道要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| roadAccess | boolean | 是 | 消防车能否到达 |
| entranceCount | int | 否 | 入口数量 |
| evacuationRouteClear | boolean | 是 | 疏散通道是否畅通 |

### 协同要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| hasCoordinator | boolean | 是 | 现场是否有负责人 |
| coordinatorContact | string | 否 | 负责人联系方式 |
| hasEmergencyPlan | boolean | 否 | 是否有应急预案 |

## 工厂仓库火灾细分场景

| 三级场景 | 描述 | 关键缺失要素 |
|----------|------|--------------|
| S1-4-1 | 普通仓库火灾 | 燃烧物类型、消防车可达性 |
| S1-4-2 | 危化品仓库火灾 | 危化品种类、是否泄漏、防化需求 |
| S1-4-3 | 厂房火灾 | 建筑结构、坍塌风险、供水条件 |
| S1-4-4 | 物流中心火灾 | 货物类型、燃烧物、蔓延速度 |

## 问题集继承结构

```
工厂仓库火灾类 = L0 + L1 普通火灾 + 高危结构专项问题集
```

### L0层（必须）

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| l0.root | "您好，这里是119消防指挥中心！请问您遇到什么紧急情况？在什么位置？" | incidentType, sceneType, district, community |
| l0.pos | "能再说一下具体位置吗？" | detailedAddress |
| l0.risk | "是否有人被困？有多少人？" | trappedCount, trappedLocation |

### L1层（普通火灾扩展）

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| l1.fire.1 | "烟是什么颜色？浓不浓？" | smokeColor, smokeOdor |
| l1.fire.2 | "火是刚着起来，正在蔓延，还是已经烧得很大？" | fireStage, fireScale |
| l1.fire.3 | "起火位置大概在哪里？" | buildingType, fireFloor |
| l1.fire.4 | "这里是什么类型的建筑？" | buildingType |
| l1.fire.5 | "一共有多少层？起火在第几层？" | totalFloors |
| l1.fire.6 | "楼梯、走廊或出口有没有被烟火堵住？" | evacuationRouteClear |
| l1.fire.7 | "现场有没有人正在灭火？是否已经断电断气？" | hasInitialFirefighting, powerCut |

### 高危结构专项问题

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| h.1 | "仓库里存放的是什么物品？有没有危化品？" | hasDangerousItem, hazardousMaterialList |
| h.2 | "现场有没有化学品、油料、气瓶或易燃易爆物？" | hazardousMaterial |
| h.3 | "消防车能不能开到仓库门口？路有多宽？" | roadAccess |
| h.4 | "楼内消防栓是否能用？水压正常吗？" | sprinklerStatus |
| h.5 | "现场有没有物业、保安或企业负责人？" | hasCoordinator, coordinatorContact |
| h.6 | "房屋有没有摇晃或裂缝？有没有听到异响？" | hasCollapseRisk |
| h.7 | "灭火用水量大不大？有没有消防水池？" | waterSupply |

## 关键缺失判断

**示例**：报警人说"XX物流仓库着火了，里面有很多货，黑烟很大"

| 已提供 | 缺失（按优先级） |
|--------|------------------|
| ✅ 位置（XX物流仓库） | ❌ 燃烧物类型（危化品？） |
| ✅ 火情（黑烟很大） | ❌ 是否有危化品 |
| ✅ 场所类型（仓库） | ❌ 消防车可达性 |
| | ❌ 是否需要防化 |
