# 新能源汽车火灾场景画像

新能源汽车火灾是交通运输火灾中的特殊类型，具有锂电池热失控、复燃风险高等特点。

---

## 场景定义

| 属性 | 值 |
|------|------|
| 场景编号 | T12-10 |
| 一级场景 | 交通运输火灾 |
| 二级场景 | 新能源汽车 |
| 风险等级 | 高 |
| 调派类型 | 特种灭火+防复燃 |

---

## 核心特征

- **锂电池热失控**：燃烧速度快、温度高
- **可能爆炸**：壳体破裂、喷射火
- **复燃风险高**：锂电池内部反应持续
- **烟雾毒性大**：含有重金属颗粒
- **灭火时间长**：需要持续冷却

## 场景画像要素

### 位置要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| district | string | 是 | 行政区划 |
| locationType | string | 是 | 位置类型：车库/充电站/路边/高速 |
| detailedAddress | string | 是 | 详细地址 |

### 车辆要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| vehicleType | string | 是 | 车辆类型：乘用车/商用车 |
| isElectric | boolean | 是 | 是否电动车 |
| batteryType | string | 是 | 电池类型：三元锂/磷酸铁锂 |

### 火情要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| fireStage | enum | 是 | 火灾阶段 |
| hasExplosion | boolean | 是 | 是否有爆炸 |
| smokeColor | enum | 是 | 烟雾颜色 |
| hasThermalRunaway | boolean | 是 | 是否热失控 |

### 处置要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| powerCut | boolean | 是 | 是否已断电 |
| hasSeepage | boolean | 是 | 是否有电解质渗漏 |
| surroundingVehicles | boolean | 是 | 周边是否有其他车辆 |
