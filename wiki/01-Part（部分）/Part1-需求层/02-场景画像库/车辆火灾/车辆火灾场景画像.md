# 车辆火灾场景画像

车辆火灾是 S1 火灾类下的重要场景类型，包括机动车、电动车、运输车辆等。

---

## 场景定义

| 属性 | 值 |
|------|------|
| 场景编号 | S1-5 |
| 一级场景 | 火灾类（S1） |
| 二级场景 | 车辆火灾 |
| 风险等级 | 中-高 |
| 调派类型 | 标准调派（特殊场景需特种灭火） |

---

## 核心特征

- **燃烧速度快**：燃油、塑料部件燃烧迅速
- **可能引发爆炸**：油箱、电池（锂电池）有爆炸风险
- **场景多样**：路边车辆、高速车辆、地下车库、充电桩
- **锂电池火灾特殊**：需干粉、泡沫等特殊灭火剂

---

## 场景画像要素

### 位置要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| district | string | 是 | 行政区划 |
| locationDescription | string | 是 | 位置描述：路边/高速/车库/充电站 |
| detailedAddress | string | 是 | 详细地址/地标 |
| roadSection | string | 否 | 高速路段信息 |

### 车辆要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| vehicleType | string | 是 | 车辆类型：轿车/货车/公交车/电动车 |
| fuelType | string | 是 | 燃料类型：汽油/柴油/电动/混动 |
| isElectricVehicle | boolean | 是 | 是否电动车 |
| batteryType | string | 否 | 电池类型：锂电池/铅酸（电动车） |

### 火情要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| fireStage | enum | 是 | 火灾阶段：初期/蔓延/大火 |
| fireScale | enum | 是 | 火势：小/中/大 |
| smokeColor | enum | 是 | 烟雾颜色 |
| hasExplosionRisk | boolean | 是 | 是否有爆炸风险 |

### 人员要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| trappedCount | int | 是 | 被困人数 |
| hasInjury | boolean | 是 | 是否有人受伤 |

### 通道要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| roadAccess | boolean | 是 | 消防车能否到达 |
| trafficStatus | string | 否 | 交通状态：拥堵/畅通 |
| needTrafficControl | boolean | 否 | 是否需要交通管制 |

## 车辆火灾细分场景

| 三级场景 | 描述 | 关键缺失要素 |
|----------|------|--------------|
| S1-5-1 | 路边车辆火灾 | 车辆类型、燃料类型、爆炸风险 |
| S1-5-2 | 地下车库车辆火灾 | 车库入口、是否有人员被困 |
| S1-5-3 | 高速车辆火灾 | 位置（公里数）、是否需要交通管制 |
| S1-5-4 | 电动车充电火灾 | 电池类型、是否在充电、爆炸风险 |
| S1-5-5 | 运输车辆火灾 | 货物类型、是否有危化品 |

## 问题集继承结构

```
车辆火灾类 = L0 极简问题集 + L1 普通火灾基础
```

### L0层（必须）

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| l0.root | "您好，这里是119消防指挥中心！请问您遇到什么紧急情况？在什么位置？" | incidentType, sceneType, district, locationDescription |
| l0.pos | "能再说一下具体位置吗？" | detailedAddress, roadSection |
| l0.risk | "是否有人被困？有没有人受伤？" | trappedCount, hasInjury |

### L1层（车辆火灾扩展）

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| l1.vehicle.1 | "是什么类型的车？汽油车还是电动车？" | vehicleType, fuelType, isElectricVehicle |
| l1.vehicle.2 | "火势大不大？有没有看到明火或爆炸？" | fireStage, fireScale, hasExplosionRisk |
| l1.vehicle.3 | "是什么颜色的烟？" | smokeColor |
| l1.vehicle.4 | "消防车能不能开到旁边？" | roadAccess |
| l1.vehicle.5 | "电动车充电的时候起的火吗？电池是什么类型？" | batteryType, isCharging |
