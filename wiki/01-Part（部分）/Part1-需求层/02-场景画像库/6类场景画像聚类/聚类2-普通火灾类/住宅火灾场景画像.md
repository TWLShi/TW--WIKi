# 住宅火灾场景画像

住宅火灾是消防接警中最常见的警情类型之一。

---

## 场景定义

| 属性 | 值 |
|------|------|
| 场景编号 | S1-1 |
| 一级场景 | 火灾类（S1） |
| 二级场景 | 住宅火灾 |
| 风险等级 | 中-高 |
| 调派类型 | 标准调派 |

---

## 场景画像要素

### 位置要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| district | string | 是 | 行政区划 |
| community | string | 是 | 小区名称 |
| detailedAddress | string | 是 | 详细地址 |
| floor | string | 是 | 楼层 |
| roomNumber | string | 是 | 房号 |

### 火情要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| fireStage | enum | 是 | 火灾阶段：初期、蔓延、大火 |
| fireScale | enum | 是 | 火势：烟、小火、大火 |
| smokeColor | enum | 是 | 烟雾颜色：白烟、黑烟 |
| burningMaterial | string | 是 | 燃烧物 |

### 人员要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| trappedCount | int | 是 | 被困人数 |
| trappedLocation | string | 否 | 被困位置 |
| hasVulnerablePeople | boolean | 否 | 是否有老人/儿童/病人 |

### 建筑要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| buildingType | string | 是 | 建筑类型：住宅楼 |
| totalFloors | int | 否 | 总层数 |
| hasBalcony | boolean | 否 | 是否有阳台 |

### 危险源要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| hasDangerousItem | boolean | 是 | 是否有燃气罐/电动车/电池 |
| gasLeak | boolean | 否 | 是否燃气泄漏 |

---

## 住宅火灾三级细分场景

| 三级场景 | 描述 | 关键缺失要素 |
|----------|------|--------------|
| S1-1-1 | 厨房着火 | 燃气情况、燃烧物类型 |
| S1-1-2 | 卧室着火 | 被困人员情况、门是否能开 |
| S1-1-3 | 电动车入户 | 电池类型、是否在充电 |
| S1-1-4 | 楼道杂物着火 | 蔓延情况、通道是否被堵 |
| S1-1-5 | 燃气泄漏引发火情 | 是否有爆炸风险、是否已关阀门 |
| S1-1-6 | 有人被困的住宅火灾 | 被困人员状态、能否逃生 |

## 住宅火灾问询流程

### L0层（必须）

```
L0.root → "您好，这里是119消防指挥中心！请问您遇到什么紧急情况？在什么位置？"
    ↓
L0.pos → "能再说一下具体位置吗？具体到几栋几单元几楼几号？"
    ↓
L0.risk → "是否有人被困？有多少人？"
```

### L1层（标准）

```
L0.fire.1 → "现在火势和烟的情况怎么样？"
    ↓
L0.fire.2 → "大概是什么东西着火了？"
    ↓
L1.fire.1 → "烟是什么颜色？浓不浓？"
    ↓
L1.fire.2 → "火是刚着起来，正在蔓延，还是已经烧得很大？"
    ↓
L1.fire.3 → "起火位置大概在哪里？"
    ↓
L1.fire.4 → "这里是什么类型的建筑？"
    ↓
L1.fire.5 → "一共有多少层？起火在第几层？"
    ↓
L1.fire.6 → "楼梯、走廊或出口有没有被烟火堵住？"
    ↓
L1.fire.7 → "现场有没有人正在灭火？是否已经断电断气？"
```

### L2层（按需）

根据建筑场景分类进入不同L2问题集：
- 高危结构类 → L2.highrisk
- 人员密集类 → L2.crowded
- 普通住宅 → 完成

## 住宅火灾关键缺失判断

**公式**：关键缺失要素 = 场景画像要求 - 报警人已提供

**示例**：报警人说"我家厨房着火了，烟很大，我妈还在里面"

| 已提供 | 缺失（按优先级） |
|--------|------------------|
| ✅ 位置（某小区） | ❌ 详细地址（楼栋单元） |
| ✅ 起火位置（厨房） | ❌ 楼层 |
| ✅ 火情（烟很大） | ❌ 门是否能打开 |
| ✅ 人员被困（母亲） | ❌ 是否有燃气罐 |
| | ❌ 是否已断电断气 |
