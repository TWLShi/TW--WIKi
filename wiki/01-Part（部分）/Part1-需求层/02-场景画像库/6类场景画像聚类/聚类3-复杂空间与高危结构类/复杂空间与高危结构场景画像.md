# 复杂空间与高危结构场景画像

复杂空间与高危结构类不是单一场景，而是一组"救援难度显著升高"的场景。包括高层建筑、地下空间、大跨度建筑、化工企业、城中村、老旧小区、厂房仓库、特殊结构建筑等。

---

## 场景定义

| 属性 | 值 |
|------|------|
| 场景编号 | S1-2 |
| 一级场景 | 火灾类（S1） |
| 二级场景 | 复杂空间与高危结构 |
| 风险等级 | 高-极高 |
| 调派类型 | 特殊调派（举高车/防化/重型装备） |

---

## 核心特征

与普通火灾相比，复杂空间与高危结构类的关键差异：
- **空间结构复杂**：垂直高度大、平面跨度大
- **进入难度大**：通道受限、消防车难以接近
- **烟气扩散路径复杂**：烟囱效应、竖向蔓延
- **供水、排烟、疏散、通信均可能受限**
- **可能存在爆炸、毒烟、坍塌等高危因素**
- **需要专业力量和特殊装备**

## 场景画像要素

### 位置要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| district | string | 是 | 行政区划 |
| community | string | 是 | 小区/场所名称 |
| detailedAddress | string | 是 | 详细地址 |
| buildingType | string | 是 | 建筑类型：高层/地下/厂房/仓库/特殊结构 |
| fireFloor | int | 是 | 起火楼层 |
| totalFloors | int | 是 | 总层数 |
| isUnderground | boolean | 是 | 是否为地下空间 |
| exitCount | int | 否 | 出入口数量 |

### 火情要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| fireStage | enum | 是 | 火灾阶段：初期/蔓延/大火 |
| smokeSpreadDirection | string | 是 | 烟气扩散方向：向上/向下/走廊 |
| stairwaySmoke | boolean | 是 | 楼梯间是否有烟气 |
| sprinklerStatus | string | 否 | 室内消防栓状态 |

### 危险源要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| hazardousMaterial | boolean | 是 | 是否有危化品 |
| hazardousMaterialType | string | 否 | 危化品类型 |
| hasExplosionRisk | boolean | 是 | 是否有爆炸风险 |
| powerCut | boolean | 是 | 是否已断电 |

### 通道可达要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| roadAccess | boolean | 是 | 消防车能否到达 |
| roadWidth | string | 否 | 道路宽度描述 |
| elevatorStatus | string | 否 | 电梯是否可用 |
| evacuationRouteClear | boolean | 是 | 疏散通道是否畅通 |

### 人员要素

| 字段 | 类型 | 必须 | 说明 |
|------|------|------|------|
| trappedCount | int | 是 | 被困人数 |
| trappedLocation | string | 是 | 被困位置（楼层/区域） |
| hasCoordinator | boolean | 否 | 现场是否有负责人/保安 |

## 复杂空间细分场景

| 三级场景 | 描述 | 关键缺失要素 |
|----------|------|--------------|
| S1-2-1 | 高层建筑火灾 | 火灾楼层、烟囱效应、被困位置 |
| S1-2-2 | 地下空间火灾 | 入口位置、出入口数量、缺氧风险 |
| S1-2-3 | 大跨度厂房火灾 | 坍塌风险、危化品情况、供水条件 |
| S1-2-4 | 化工企业火灾 | 危化品种类、爆炸风险、防化要求 |
| S1-2-5 | 老旧小区/城中村火灾 | 消防车可达性、通道宽度、违章情况 |

## 问题集继承结构

```
复杂空间与高危结构类 = L0 + L1 普通火灾 + 高危结构专项问题集
```

### L0层（必须）

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| l0.root | "您好，这里是119消防指挥中心！请问您遇到什么紧急情况？在什么位置？" | incidentType, sceneType, district, community |
| l0.pos | "能再说一下具体位置吗？具体到几栋几单元几楼几号？" | detailedAddress, landmark |
| l0.risk | "是否有人被困？有多少人？" | trappedCount, trappedLocation |

### L1层（普通火灾扩展）

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| l1.fire.1 | "烟是什么颜色？浓不浓？" | smokeColor, smokeOdor |
| l1.fire.2 | "火是刚着起来，正在蔓延，还是已经烧得很大？" | fireStage, fireScale |
| l1.fire.3 | "起火位置大概在哪里？" | fireFloor, burningMaterial |
| l1.fire.4 | "这里是什么类型的建筑？" | buildingType |
| l1.fire.5 | "一共有多少层？起火在第几层？" | totalFloors, fireFloor |
| l1.fire.6 | "楼梯、走廊或出口有没有被烟火堵住？" | evacuationRouteClear, stairwaySmoke |
| l1.fire.7 | "现场有没有人正在灭火？是否已经断电断气？" | hasInitialFirefighting, powerCut |

### 高危结构专项问题

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| h.1 | "烟是往楼上、楼下，还是向走廊扩散？" | smokeSpreadDirection |
| h.2 | "烟囱效应明显吗？楼梯间能看到烟吗？" | stairwaySmoke |
| h.3 | "楼内消防栓是否能用？水压正常吗？" | sprinklerStatus |
| h.4 | "地下空间入口在哪里？有几个出入口？" | isUnderground, exitCount |
| h.5 | "现场有没有化学品、油料、气瓶或易燃易爆物？" | hazardousMaterial, hazardousMaterialType |
| h.6 | "消防车能不能开到楼下或门口？路有多宽？" | roadAccess, roadWidth |
| h.7 | "现场有没有物业、保安或企业负责人？" | hasCoordinator |
| h.8 | "房屋有没有摇晃或裂缝？有没有听到异响？" | hasCollapseRisk |
| h.9 | "现场是厂房、仓库、地下车库还是商场中庭？" | buildingType |

## 关键缺失判断

**公式**：关键缺失要素 = 场景画像要求 - 报警人已提供

**示例**：报警人说"XX大厦20楼着火了，楼梯都是烟，有人还在里面"

| 已提供 | 缺失（按优先级） |
|--------|------------------|
| ✅ 位置（XX大厦20楼） | ❌ 总层数 |
| ✅ 建筑类型（高层） | ❌ 火灾阶段 |
| ✅ 楼梯有烟（逃生受阻） | ❌ 燃烧物类型 |
| ✅ 人员被困 | ❌ 烟囱效应程度 |
| | ❌ 消防车可达性 |
| | ❌ 是否已断电 |
