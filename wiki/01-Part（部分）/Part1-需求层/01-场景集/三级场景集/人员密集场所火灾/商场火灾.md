# S1-9-1 商场火灾

三级场景编号：S1-9-1

---

## 场景定义

| 属性 | 值 |
|------|------|
| 所属一级 | S1 火灾类 |
| 所属二级 | S1-9 人员密集场所火灾 |
| 三级场景 | 商场火灾 |
| 风险等级 | 高-极高 |
| 调派类型 | 大规模人员+医疗联动 |

---

## 场景特征

- 人员密集，疏散难度大
- 空间大，火灾蔓延快
- 商品多，燃烧物复杂
- 可能引发踩踏等次生灾害

---

## 关键画像要素

| 要素类别 | 必须字段 | 说明 |
|----------|----------|------|
| 位置 | district, community, fireFloor | 着火楼层 |
| 人群 | estimatedPeopleCount, crowdType | 人数+人群类型 |
| 火情 | fireStage, smokeSpreadDirection | 火势+蔓延方向 |
| 疏散 | evacuationStatus, exitBlocked | 疏散状态+出口 |
| 人员 | trappedCount, hasVulnerablePeople | 被困+老幼病残 |

## 关键缺失要素

| 缺失项 | 等级 | 影响 |
|----------|------|------|
| 现场大概人数 | A级 | 疏散规模判断 |
| 已撤离/未撤离人数 | A级 | 救援优先级 |
| 出口是否畅通 | A级 | 疏散通道判断 |
| 有没有人受伤 | B级 | 联动医疗 |
| 是否有负责人组织 | D级 | 影响现场协同 |
