# 暴雨内涝被困

## 场景定义

| 属性 | 值 |
|------|------|
| 场景编号 | S1-1 |
| 一级场景 | 水域灾害类（S1） |
| 二级场景 | 暴雨内涝 |
| 风险等级 | 中-高 |
| 调派类型 | 防汛抗洪+舟艇装备 |

---

## 场景特征

- **突发性强**：短时强降雨导致积水快速上涨
- **人员被困**：车辆或低洼区域人员无法自行撤离
- **二次伤害风险**：触电、溺水、建筑物倒塌
- **通道受阻**：道路积水导致救援车辆难以接近

---

## 关键缺失要素

| 缺失要素 | 优先级 | 说明 |
|----------|--------|------|
| waterDepth | P1 | 水深信息，直接影响救援方式 |
| trappedCount | P1 | 被困人数 |
| waterLevelTrend | P2 | 水位是否还在上涨 |
| rescueAccessible | P2 | 救援通道是否可通行 |

## 问题集（L0 + 水灾专项）

### L0层（必须）

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| l0.root | "您好，这里是119消防指挥中心！请问您遇到什么紧急情况？在什么位置？" | incidentType, district, community |
| l0.pos | "能再说一下具体位置吗？附近有什么标志建筑？" | detailedAddress, landmark |
| l0.risk | "有没有人员被困？有多少人？" | trappedCount |

### 水灾专项

| 节点 | 问题 | 提取字段 |
|------|------|----------|
| w.1 | "水有多深？到膝盖、腰部还是更高？" | waterDepth |
| w.2 | "水位还在上涨吗？水流急不急？" | waterLevelTrend, waterFlowSpeed |
| w.3 | "有没有电线落水或电器泡在水里？有没有闻到燃气味？" | hasElectrocutionRisk, hasGasLeak |
| w.4 | "被困人员能自己转移到高处吗？还是完全无法移动？" | canSelfRescue |
| w.5 | "救援车辆能开过去吗？还是需要船？" | rescueAccessible, needBoat |

## 典型对话

**报警人**："喂，119吗？外面下大雨，我们小区地下车库被淹了，有个人困在里面出不来！"

**系统**（l0.root）："您好，这里是119消防指挥中心！请问您遇到什么紧急情况？在什么位置？"
→ 提取：incidentType=水域灾害，community=XX小区

**系统**（l0.pos）："能再说一下具体位置吗？地下车库入口在哪里？"
→ 提取：location=地下车库入口

**系统**（l0.risk）："有没有人员被困？有多少人？"
→ 报警人："有1个人，卡在车里了"

**系统**（w.1）："水有多深？到膝盖、腰部还是更高？"
→ 报警人："已经到胸口了"

**系统**（w.2）："水位还在上涨吗？"
→ 报警人："还在涨，雨太大了"

**系统**（w.3）："有没有电线落水或闻到燃气味？"
→ 报警人："没有"

**系统**（w.4）："那个人能自己爬到车顶吗？"
→ 报警人："不行，水流太急，门都打不开"

**系统**（w.5）："消防车能开到车库门口吗？"
→ 报警人："进不来，外面路面积水也很深"

→ 调派：防汛抗洪车+舟艇，紧急救援
