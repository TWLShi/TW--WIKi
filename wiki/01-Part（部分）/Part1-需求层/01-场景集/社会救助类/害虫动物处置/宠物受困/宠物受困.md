# S3-4-3 宠物受困（高楼、夹缝等）

三级场景编号：S3-4-3

---

## 场景定义

| 属性 | 值 |
|------|------|
| 所属一级 | S3 社会救助类 |
| 所属二级 | S3-4 害虫动物处置 |
| 三级场景 | 宠物受困（高楼、夹缝等） |
| 风险等级 | 低-中 |
| 调派类型 | 动物救援+高空作业（必要时） |

---

## 场景特征

- 宠物被困高处或夹缝
- 动物可能受伤或恐慌
- 可能需要高空作业
- 以救援为主，非紧急

---

## 关键画像要素

| 要素类别 | 必须字段 | 说明 |
|----------|----------|------|
| 位置 | district, community, detailedAddress, height | 位置/高度 |
| 宠物 | petType, petSize, petCondition | 宠物类型/大小/状况 |
| 环境 | trappedLocation, accessRoute, hasFood | 受困位置/进入路线/食物 |
| 人员 | ownerPresent, canCalmPet | 主人是否在场/能否安抚 |
| 时间 | trappedDuration, hasWater | 受困时长/是否有水 |

---

## 关键缺失要素

| 缺失项 | 等级 | 影响 |
|----------|------|------|
| 宠物位置和高度 | A级 | 救援方案 |
| 宠物类型和大小 | B级 | 装备选择 |
| 是否有坠落风险 | A级 | 紧迫度 |
| 进入路线 | B级 | 处置难度 |
| 主人配合度 | C级 | 处置顺利度 |
